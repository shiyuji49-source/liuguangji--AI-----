# 三引擎价格对比 + 接入定稿

> ## ✅ 图像网关定稿：DMXAPI（2026-06-15，覆盖下文 OpenRouter/乐奇 讨论）
> 用户裁定图像走 **DMXAPI**（base `https://www.dmxapi.cn`，Bearer 鉴权，一个 DMXAPI key 管两个模型）。乐奇已排除（无这两模型）；OpenRouter 仅作备选。视频仍走火山方舟。
> **⚠️ 两个图像模型在 DMXAPI 上是两套不同协议，provider 层要写两个适配器：**
>
> ### A. image2 = gpt-image-2 —— OpenAI Images 协议
> - 文生图：`POST https://www.dmxapi.cn/v1/images/generations`
> - 图生图/编辑：`POST https://www.dmxapi.cn/v1/images/edits`（multipart，字段 `image` 可多张 + 可选 `mask`<50MB）
> - `model`: **`gpt-image-2-ssvip`**（官方推荐稳定版；或 `gpt-image-2`）
> - 参数：`prompt`(≤32000字)、`n`(1-10)、`size`(1024x1024 … 最大 4K 3840x2160)、`quality`(**auto/high/medium/low** ← 对应低/中/高)、`output_format`(png/jpeg/webp)、`output_compression`(0-100)、`background`(auto/opaque)、`moderation`(auto/low)
> - **档位映射**：低=quality:low / 中=quality:medium / 高=quality:high；分辨率另由 size 控（1K/2K/4K → 1024²/2048²/4096² 或对应宽高比）
> - 返回：`data[].b64_json` 或 `data[].url`
>
> ### B. nano banana pro = gemini-3-pro-image —— Google Gemini 协议
> - 全部走：`POST https://www.dmxapi.cn/v1beta/models/gemini-3-pro-image:generateContent`（文生图/编辑/多图融合/多轮 同一端点，靠 payload 区分）
> - `model`: **`gemini-3-pro-image`**（别名 Nano Banana Pro）
> - **档位**：`generationConfig.imageConfig:{ "aspectRatio":"16:9", "imageSize":"1K"|"2K"|"4K" }`（1K=1024²/2K=2048²/4K=4096²；比例支持 1:1/16:9/9:16/2:3/3:2/3:4/4:3/4:5/5:4/21:9）；`responseModalities:["IMAGE"]`
> - 参考图：`parts[].inline_data:{mime_type,data:base64}`；多图融合上限 **6 主体图 + 5 角色图**
> - 多轮：保留并回传 `thoughtSignature`（模型推理链，下一轮原样带回）
> - 返回：`candidates[0].content.parts[].inlineData.data`（base64）。⚠️**坑**：有时把 base64 当 text 直接返回（`parts[].text="data:image/png;base64,..."`），provider 层要两种都兜。
> - DMXAPI 计费（token）：1K/2K=1210 tokens、4K=2000 tokens。
>
> **本平台落地**：`src/lib/ai/image.ts` 写两个适配器（OpenAI 协议 + Gemini 协议），共用 DMXAPI base+key；档位→参数按上表；返回统一解码 base64 → storage.put → assets 墙。`.env` 加 `DMXAPI_API_KEY`/`DMXAPI_BASE_URL=https://www.dmxapi.cn`。**DMXAPI 实际单价以其后台/充值页为准，重定价见 §0.1。**

# 三引擎价格对比：官方 vs OpenRouter vs 乐奇（2026-06 调研，网关已改 DMXAPI）

> 换算口径：1 USD ≈ ¥7.2；本平台 1 元 = 100 积分。价格随时变，**上线前以各家控制台当日价为准**。乐奇为私有代理，网上无公开价，数字需你从乐奇后台拿。

## 0. 一句话结论（先看这个）

1. **⚠️ 最重要：现行图片售价档（24/36/48 积分）是按便宜模型设的，远低于 gpt-image-2 / nano banana pro 的真实成本——高清档成本就要 150-170 积分。不改价＝每张倒贴。必须上线前重定价。**
2. 三个网关里，**OpenRouter ≈ 官方价 + 约 5.5% 充值手续费**（几乎等于官方价），透明、一个 key 管两个图片模型。
3. **乐奇价格只有它后台知道**——代理商常见两种：比官方便宜（走优惠通道）或加价换 RMB 结算便利；要你提供。
4. **国内可达性才是选网关的真正关键**：官方 OpenAI/Google 在国内服务器直连会被墙，所以才要走 OpenRouter 或乐奇——但 **OpenRouter 在国内服务器能不能稳连也要实测**（它是美国服务）。乐奇（lqapi.top）国内可达但我们已知它多上游不稳。
5. **视频（Seedance）没有 OpenRouter**，直连火山方舟即可（国内、RMB、~¥1/秒），现行视频售价档（6.9/7.65 积分/1k token）≈ 1.5 倍加价，基本合理。

## 6. 国内聚合平台选型（关键方向决策）

**铁律：「大、稳、合规、国内」与「用 gpt-image-2 / nano banana pro 外国模型」基本互斥。** 二选一：

### 路线 A：坚持外国图像模型 → 只能用「中转站」
国内大平台（火山/阿里/硅基流动）官方不代理 OpenAI/Google 图像模型；能拿到的都是中转站，乐奇是其中较小一个。比乐奇更大/更稳的中转站候选（仍属同一灰色品类，随时可能断供/涨价，**价格在各家后台、需注册查**）：
- **302.ai**——较知名、企业向、OpenAI 兼容、按量充值，模型全。
- **DMXAPI / PoloAPI(poloapi.top) / proaiapi / n1n.ai / 非线智能**——生产级稳定性口碑较好（[盘点](https://www.oschina.net/news/410911)、[选型](https://zhuanlan.zhihu.com/p/1995514752835810220)）。
- 选法：挑 2 家注册→各充小额→压测连通率/延迟/断供率→看图片单价是否优于 OpenRouter。本质都是"官方价×折扣或加价"，**没有公开定价，要后台查**。
- 风险：外国模型经中转站＝合规与供给都不在自己手里，企业内测可，规模化有风险。

### 路线 B（推荐认真考虑）：图像换国产 → 用一个大平台搞定图+视频
**火山方舟（你已有 ARK key）一个平台同时有：Seedream 5.0（即梦，文生图/图生图）+ Seedance 2.0（视频）。** 一个 key、国内直连、RMB、合规、稳定，图+视频统一。代价＝放弃 gpt-image-2/nano banana，改用即梦 Seedream（对中文短剧/古装资产其实很对口，质量国内第一梯队）。
- 备选大平台：**硅基流动**（开源+国产图像模型多、OpenAI 兼容、价低、[价目](https://www.siliconflow.cn/pricing)）；**阿里百炼**（通义万相 Wanxiang 图像）。
- 这条路把"网关稳定性"问题直接消灭——不再依赖任何中转站或海外服务。

### 建议
- 若把"稳定可控、合规、国内可达"放第一位 → **路线 B（火山方舟 Seedream + Seedance 一站式）最优**，且你已有 key、video 本来就走它。
- 若一定要 gpt-image-2 / nano banana 的特定效果 → 路线 A，从 302.ai / DMXAPI 里挑一个比乐奇大的，或继续用 OpenRouter（先测国内连通性）。
- 折中：**视频用火山 Seedance（已定）；图像可双轨**——高级档用 nano banana（经 OpenRouter/中转站），走量档用火山 Seedream（国内稳、便宜），admin 定价表分别定价。架构上 image provider 做成可多后端即可。
