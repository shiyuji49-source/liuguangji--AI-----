# 鎏光智绘 · AI 影视剧制作 SOP

> **给谁看**：新入职的导演 / 导演助理（分镜师）/ 美术师 / 后期。看完你就知道——项目怎么走、**你这一环干什么、用什么软件工具、交付什么、过哪道审核**。
> **本文不讲提示词 skill 怎么用**（那 8 个 skill 是制作中要用到的工具，用法见《鎏光智绘提示词SKILL使用说明》）。本文只告诉你"哪一步用哪个工具、按什么规范做"。

## 一、团队角色与职责

| 角色 | 一句话职责 |
|---|---|
| **导演** | 对项目**全权负责**。立项拉片、分工、定风格/镜头/重要提示词，向内容负责人过审，审资产入库，盯全流程质量。 |
| **导演助理（分镜师）** | 分镜与镜头规划；出静帧、出视频提示词、生成视频。 |
| **美术师** | 做各类资产图（人物/服装/道具/场景/群演），做智能超清，送审入库。 |
| **后期** | 剪辑、调色、音效、字幕、成片归档。 |

## 三、全流程总览

```
立项（拉片 + 全员通读剧本 + 导演分工 + 向内容负责人过审）
        ↓
资产制作（MJ 定调 → skill 写提示词 → image2 出图 → 即梦超清 → 导演审核 → 入库 NAS）
        ↓
视频生成（分镜大师出静帧 → Seedance 出视频，多图参考）
        ↓
后期（剪辑 → 调色 → 音效 → 字幕 → 成片）→ 归档 NAS
```

> **贯穿原则**：第一集向上拔一档质感；@资产名全程不变；分级（B/A/S）决定静帧与视频做法（细节见《使用说明》）。

## 四、各阶段规范

### 1. 项目立项（导演主导，全员参与）
- 开**立项会拉片**；**全员通读剧本**。
- 导演**分工**：谁改剧本、谁做哪类资产、谁分镜出视频、谁后期。
- 导演向**公司内容负责人**阐述：**制作思路、镜头规划、风格定调、重要提示词**——**过审后才能进入制作**。
- 确定项目**分级 B / A / S**，写进项目头（影响静帧取舍与视频骨架）。
- **第一集向上拔一档**：第一集是门面（定完播、定过审），用**高于本片定级一档**的质感做。例：B 级片第一集按 A 级质感、A 级片第一集按 S 级质感。

### 2. 资产制作（美术师）
- 按导演分工领资产线（人物 / 服装 / 道具 / 场景 / 群演）。
- **前期风格参考**：可用 **MJ** 搜索 + 生成，先把人物/场景的美学方向定下来。
- **写提示词**：用对应提示词 skill（**首选 Claude**；用法见《使用说明》）。
- **出图**：默认 **image2**——简单图 **instant**，复杂图 & 二次溶图 **think**；二次溶图也可用 **Gemini nano banana pro（在 flow 中）**。
- **智能超清（必做）**：所有资产图都要超清——**即梦画布·智能超清，细节 < 5（防突变），分辨率 ≤ 7000px**。
- **质量红线**：**禁 AI 感、禁脏、禁油**（连同糊、畸变、穿模一并卡）。
- **送审入库**：资产**必须经导演审核通过，方可入库**；入库后才能用于视频。命名 + 分类存入 NAS（见第七节）。

### 3. 视频生成（导演助理 / 分镜师）
- 用 **分镜大师** 出关键帧静帧提示词 → image2（+ 已入库资产）出**静帧**（S/A 级选择性出、B 级可选）。静帧同样要超清、守红线。
- 用 **Seedance** 出视频提示词 → Seedance（**多图参考**：资产母版 + 选择性静帧）→ 强控视频。
- 视频生成端：**禁配乐、禁字幕、禁水印、禁反射镜头**；血腥统一处理为暗红污渍/侧面构图。

### 4. 后期（后期）

- **剪辑**：按分镜节奏拼接；善用 **L-cut**（声音先入或延续到下一镜）；掉帧/顿挫镜头必要时逐帧删重复帧后只当 B-roll；保留每集开头钩子与结尾扣子。
- **调色**：**全片统一基调**，参考资产/静帧的色温与配色（如 60:30:10 主/辅/点缀），同一场景内不跳色；建议套**统一 LUT**，先定一集为基准再套全剧。
- **音效**：环境音 + 动效到位；**配乐在后期加**（生成端不加）；对白响度统一，留好声场层次。
- **字幕**：**生成端不烧字幕，字幕在后期统一加**；字体/字号/位置/安全边距按公司模板；竖屏注意避开顶/底安全区。
- 成片导出 + 归档 NAS。

## 五、三道审核关（卡点，过不了不往下走）

1. **进入制作前** —— 导演向**内容负责人**过：制作思路 / 镜头规划 / 风格定调 / 重要提示词。
2. **资产入库前** —— **导演**逐张审资产：无 AI 感 / 脏 / 油，超清达标（细节<5、≤7000px），风格统一。**未过审不得入库、不得进视频。**
3. **成片前** —— 导演终审（必要时加内容负责人）：节奏、调色、音效、字幕、过审风险。

---

## 六、质量红线（贴墙版）

- **资产 / 静帧**：禁 **AI 感 / 脏 / 油 / 糊 / 畸变 / 穿模**；必做智能超清，**细节 < 5、分辨率 ≤ 7000px**。
- **第一集向上拔一档**质感。
- **@资产名全流程不变**，所有人用同一份资产清单命名与调用。
- **生成端**：禁配乐、禁字幕、禁水印、禁反射镜头；血腥→暗红污渍、明星不 1:1 还原、品牌虚构。
- **未过导演审核的资产不入库、不进视频。**

---

## 七、归档规范（NAS）

- **提示词、资产、视频**全部按**公司资产整理规范**分类放入 NAS，不留在个人电脑。
- **资产文件名**：`@资产名_状态.png`（与资产清单一致），例：`杨延昭_定妆照.png`、`金殿_日景.png`。
- **建议项目目录**：
  ```
  项目名/
    00_剧本/        影视化剧本 + 资产清单
    01_资产/        人物 / 服装 / 道具 / 场景 / 群演（均为超清入库版）
    02_静帧/        第X集
    03_视频/        第X集
    04_成片/
    05_提示词/      各阶段用过的提示词留档
  ```

---

## 八、新人 30 秒上手（对号入座）

- **美术师** → 看第二节工具 + 第四节·2 + 第六节红线。流程：领资产 → MJ 定调 → skill 写提示词（Claude）→ image2 出图（instant/think，溶图可 Gemini nano）→ 即梦画布超清（细节<5、≤7000px）→ **送导演审** → 入 NAS。
- **导演助理 / 分镜师** → 第四节·3。流程：分镜大师出静帧 → Seedance 出视频（多图参考）→ 交后期。
- **后期** → 第四节·4 + 第六节红线。剪辑 → 调色（统一 LUT）→ 音效（配乐后期加）→ 字幕（后期加）→ 成片归档。
- **导演** → 第四节·1 + 第五节审核关。你对项目负责：拉片分工、向内容负责人过审、审资产入库、终审成片。
