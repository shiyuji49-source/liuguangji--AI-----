# Image Prompt Director Alignment

本 Skill 生成的每条场景提示词都必须符合 image-prompt-director 的视觉口径。

## 必须具备

- 一个明确视觉中心，而不是平均写满所有建筑和道具。
- 一个观看位置：摄影机在哪里、离场景多远、从什么高度看。
- 清楚空间层级：前景、中景、远景、背景各自承担什么。
- 主光来源明确：窗光、门洞光、烛火、天光、街灯、火盆、清晨斜光、阴天环境光等。
- 颜色落在真实载体上：木门、红漆、青石、米白纸窗、暗金屏风、湿地、旗帜、布幔、玻璃、霓虹、雪地等。
- 成像介质克制：电影写实、35mm 胶片质感、数字电影机响应、真实材质粗糙度、柔和高光滚降、干净暗部。
- 少数失败防护：避免主要人物、角色海报感、CG 塑料感、过度 HDR、满屏均匀高清、现代违和物件等。

## 避免

- 只写“豪华、震撼、电影感、超高清”的空泛质量词。
- 把场景写成游戏地图、旅游宣传图、建筑渲染或道具陈列。
- 无来源轮廓光、蓝金网游光、过量烟雾、过量粒子、全画面同等清晰。
- 把所有陈设、纹样、摊位和建筑都写得一样重要。
- 因为剧本里有角色，就默认把角色写进画面。

## 场景提示词的电影化方向

场景应像摄影机提前或滞后于事件半步到达现场：人物不在，但空间还保留刚刚发生或即将发生的压力。

例如：

- 审判场景：不写审判者，写屏风、茶案、文书、光线压迫与空地距离。
- 战后场景：不写主将，写旗帜倾斜、车辙、火盆余烬、地面痕迹与阴天压迫。
- 街市场景：不写主角，写人群让出的空线、摊位生活证据、远景城门与清晨烟火气。
- 密谈场景：不写谈话双方，写半开的门、压低的窗光、两只茶盏、一处空座和暗部。
