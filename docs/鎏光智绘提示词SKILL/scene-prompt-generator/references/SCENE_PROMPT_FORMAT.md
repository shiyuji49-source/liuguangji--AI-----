# Scene Prompt Format

场景提示词必须是可直接复制到 AI 生图工具的机器提示词。它不是场景分析，也不是美术说明书。

## 核心顺序

1. 场景身份：时代、地域、地点、场景状态。
2. 观看位置：摄影机从哪里看，朝向哪里，景别如何。
3. 空间层级：前景、中景、远景、背景如何分配。
4. 场景资产：建筑、陈设、材质、地面、墙面、门窗、器物、标识、自然环境。
5. 事件痕迹：人物不出现，但空间保留剧情压力。
6. 光线与空气：主光来源、方向、软硬、天气、雾、烟、尘、雨、雪。
7. 颜色载体：颜色必须落在可信物体上，不做抽象滤镜。
8. 成像介质：电影写实、数字电影机、胶片质感、镜头焦段、景深控制。
9. 避免项：只写少数最相关失败防护。

## 推荐句式

```text
[时代/地域/地点]，[场景状态]，无主要人物入画。摄影机从[机位位置]以[景别/焦段/高度]观察，[空间轴线/观看方向]成为画面中心，[前景/中景/远景]形成清楚层次。

前景保留[材质/地面/门框/器物/事件痕迹]，中景是[核心建筑/陈设/空间结构]，远景以[城墙/山体/街巷/屏风/窗格/廊柱]退入空气层。场景细节集中在[视觉中心]，其他摊位、梁柱、纹样或杂物降低可读性。

[主光]从[方向/来源]进入，照亮[颜色载体]，暗部保持干净，[空气介质]让远处层次柔和退后。电影写实质感，克制数字电影机响应/35mm 胶片质感，真实材质粗糙度与接触阴影，避免[少数失败项]。
```

## 场景类型提示

### 街市 / 城门 / 巷道

重点写街道轴线、摊位层次、招幌、地面材质、远景城门、生活痕迹、光线方向。默认不写主角，必要时写远景无脸百姓剪影。

### 宫殿 / 偏殿 / 朝堂

重点写轴线、台阶、柱列、屏风、帷幔、案几、门洞光、权力尺度。避免把皇帝或官员写进画面，改写为空座、屏风剪影、茶案余温、文书印泥等。

### 室内 / 书房 / 客栈 / 茶楼

重点写入口视角、窗格光、桌案、屏风、墙面、地面、器物、生活证据。避免人物表演，改用未收茶盏、微乱帘幕、翻开的书、残烛。

### 军营 / 战场 / 边关

重点写营帐队列、旗帜、雪地/泥地、火盆、兵器架、车辙、远山、压迫天气。默认不出现将领，必要时只保留远处无脸队列或空营状态。

### 山水 / 荒原 / 寺院 / 武侠场景

重点写留白、地形、门洞、竹林、湖岸、石阶、雾、风向、衣袍痕迹或器物痕迹。避免满屏飞叶、网游光和山水壁纸感。

### 现代都市 / 办公室 / 商业空间

重点写真实地点表面、玻璃、灯管、办公桌、会议室、走廊、雨地反射、屏幕冷光和现场色偏。避免广告棚拍、过度整洁与泛泛商务素材感。
