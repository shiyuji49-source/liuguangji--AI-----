# Output Examples

## 示例 1：年代街市场景

输入剧本片段：

```text
1-1 皇城大街 日/外
汴京街头，人来人往。孩童在巷口唱童谣。远处一个征尘残破的人影走来，路人惊恐避让。
```

输出：

```text
## S01-001 汴京皇城大街 · 日外 · 清晨街市异动
【场景用途】建立汴京繁华市井，并保留远处异样来临前的空间压力。
【提示词】
北宋汴京皇城大街清晨，空景场景图，无主要人物入画。摄影机从街道一侧檐下以眼平中远景望向远处城门，宽阔青石街道形成纵深轴线，前景茶肆幌子与挑担摊位轻微遮挡画面边缘，中景巷口与街心空线成为视觉中心，远景朱楼屋脊和皇城城门退入薄雾。

街面保留真实市井痕迹：未收的糖人摊、翻卷彩帛、木桌竹篮、湿润石板、被匆忙推开的摊布与一串不连续泥水脚印。两侧可以有极远景无脸百姓剪影退到檐下，只作为尺度，不出现演员正脸或角色表演。

清晨暖金斜光从右上方穿过薄雾与炊烟，照亮红木招牌、米白布幌和青灰石板，暗部保持干净。35mm 电影写实质感，轻微胶片颗粒，真实木料与布料粗糙度，避免主要人物、角色海报感、现代物件、CG 塑料感、满屏均匀高清和过量烟雾。
```

## 示例 2：宫中偏殿场景

输入剧本片段：

```text
1-3 宫中偏殿 日/内
神秘人在屏风后泡茶，王诜跪在屏风前听判。茶汤落盏，判决已成。
```

输出：

```text
## S01-002 宫中偏殿 · 日内 · 屏风茶审后
【场景用途】表现权力隐藏在屏风后的轻描淡写与审判余威。
【提示词】
北宋宫中偏殿日内空景，无主要人物入画。摄影机站在殿门口以平视广角望向室内深处，前景是大面积青石地砖与门槛阴影，中景一组三折水墨山水绢面屏风占据画面中心，屏风两侧留出狭窄通道，远景后墙卷轴和低矮茶案被压在暗部。

屏风前的茶案上留有刚点过茶的白瓷盏、茶筅、微湿茶痕、半卷文书和一方暗红印泥，地面有一处单人跪坐后形成的微暗压痕。陈设克制，暗木案几、素色帘幕、青铜香炉与屏风绢面承担权力气氛，不展示官员正脸、不出现角色表演。

日间窗格光从画面左侧斜切进来，被屏风绢面柔化，茶盏边缘与印泥获得小面积高光，殿内暗部沉稳干净。克制数字电影机响应，柔和高光滚降，真实石材、绢面和木漆质感，避免舞台布景感、网游蓝金光、过量烟尘、主要人物入画和宫殿旅游宣传图感。
```

## 示例 3：现代办公室场景

```text
## S02-006 总裁办公室 · 夜内 · 危机会议后
【场景用途】表现商业危机爆发后留下的冷硬压力。
【提示词】
现代高层总裁办公室夜内空景，无主要人物入画。摄影机从会议桌末端以低位中远景看向落地窗，长桌、空椅和城市夜景形成冷静纵深，前景一角有翻开的合同、停止播放的投影遥控器和半杯冷咖啡，中景玻璃桌面反射天花灯带，远景高楼灯点被雨痕切碎。

办公室保持真实商业空间表面：深灰地毯、黑色皮椅、金属桌脚、玻璃隔断、墙面隐藏灯槽和一块熄屏电视。场景细节集中在桌面文件与空椅秩序，其他装饰主动退后，不出现角色、正脸或摆拍人物。

窗外雨夜城市冷蓝光与室内低照度暖白灯混合，玻璃和桌面承担主要颜色反射，暗部干净不过黑。现代数字电影机质感，轻压缩空间，真实雨痕与接触阴影，避免广告棚拍、过度整洁、HDR 高饱和、人物海报感和假景观合成感。
```
