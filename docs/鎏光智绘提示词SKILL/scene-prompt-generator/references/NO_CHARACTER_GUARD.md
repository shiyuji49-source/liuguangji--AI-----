# No Character Guard

场景图默认不包含主要人物。场景必须靠空间、光线、陈设和事件痕迹成立。

## 必须删除

- 角色姓名。
- 主角、反派、重要配角、皇帝、将军、官员、男女主、师父、师兄、刺客等明确人物。
- 正脸、半身、面部特写、表演、情绪眼神、台词动作。
- 角色服装细节、妆发、武器持握姿态。

## 可以保留

- 空椅、空案、空街、空殿、空车厢、空牢房。
- 人物离开后的痕迹：脚印、水痕、掀开的帘、散落文书、未收茶盏、倒翻椅凳、熄灭灯烛、被推开的门。
- 远景无脸群演作为尺度：街市百姓剪影、军营队列影子、酒楼食客模糊背影。必须低频、无姓名、无正脸、不抢视觉中心。
- 屏风后、帘幕后、门缝里可以出现抽象剪影，但不能写具体角色身份，除非用户要求。

## 推荐控制语

- `无主要人物入画`
- `空景场景图`
- `没有演员正脸，没有角色表演`
- `只保留远景无脸人群作为尺度`
- `人物存在感转化为环境痕迹`
- `场景细节承担叙事，不做人物海报`

## 转换示例

原剧情：`杨延昭拖着伤腿走过皇城大街，百姓惊恐避让。`

场景化：`汴京皇城大街清晨空景，街心石板上留有长距离拖痕与泥水脚印，两侧摊位帘布微乱，远处无脸百姓剪影退到檐下，街道中央形成一条被让开的空线。`

原剧情：`神秘人在屏风后泡茶，王诜跪在屏风前听判。`

场景化：`宫中偏殿日内空景，水墨绢面屏风占据中景，屏风前茶案上茶盏仍有热气，案边文书、印泥和压痕形成审判后的余威，青石地面留下单独跪坐过的压暗痕迹。`

原剧情：`男女主在雨夜便利店门口争吵。`

场景化：`雨夜便利店门口空景，玻璃门半开，地面雨水反射店内冷白灯，门口两把被丢下的伞和一只倾倒纸杯暗示刚刚发生过争执。`
