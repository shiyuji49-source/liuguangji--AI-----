# Scene Extraction

从剧本中提取场景时，优先识别“可被美术部门制作和复用的空间”，而不是逐句剧情。

## 识别来源

常见剧本格式包括：

- `1-1 皇城大街 日/外`
- `第3集 场景5 王府书房 夜 内`
- `INT. OFFICE - NIGHT`
- `外景 / 山道 / 黄昏`
- 动作段落中隐含的地点，例如“屏风后”“城门下”“酒楼二层雅间”。

若文档没有标准场次标题，则根据地点变化、时间变化和空间描述自行切分。

## 场景命名

统一命名应清楚表达：

`集数-序号 + 地点核心 + 时间/状态`

示例：

- `S01-001 汴京皇城大街 · 日外 · 清晨街市基础版`
- `S01-002 汴京皇城大街 · 日外 · 戒严后的空街`
- `S01-003 宫中偏殿 · 日内 · 屏风审判茶案`
- `S02-004 北境军营 · 夜外 · 雪夜巡防`

## 合并与拆分

同一物理空间可以有多个“场景状态”。状态变化由以下因素决定：

- 时间：清晨、正午、黄昏、夜、雨夜、雪夜。
- 天气：雾、雨、雪、风沙、炊烟、尘土、湿冷、燥热。
- 陈设：宴席、审判、祭祀、战备、搬空、封锁、被翻乱。
- 剧情后果：战后、火灾后、水淹后、倒塌后、血迹清理后、庆典结束后。
- 权力气质：朝会、私审、密谈、等候、囚禁、监视、逃亡。
- 视角资产：正视角、反向视角、入口望内、内望门口、鸟瞰、走廊纵深。

如果只是角色在同一地点交谈，不拆为新场景。

## 剧情痕迹转场景痕迹

把人物动作转换为空间痕迹，而不是把人物写入画面。

- “将军拖着断枪入城” → 石板路上的拖痕、泥水脚印、被人群让开的空线、街摊帘布微乱。
- “官员在屏风后审判” → 屏风、茶案、未散热气、压低的窗光、空椅、文书与印泥。
- “众人逃散” → 倒翻凳子、散落竹篮、未收摊布、半开的门。
- “夜里追捕” → 湿巷、脚印、水洼反光、墙面火把光、门缝暗影。
- “宴会冲突后” → 残席、倾倒酒盏、皱乱桌布、烛泪、空席位。

## 忽略项

除非用户要求，不提取以下内容为场景图主体：

- 角色面部、表演、服装特写。
- 单个可独立制作的道具展示图。
- 只有一句台词、无空间变化的过场。
- 已经被另一个场景完全覆盖的重复地点。
