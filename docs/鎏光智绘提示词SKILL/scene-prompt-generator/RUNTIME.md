# Runtime Specification

## 目标

将多集影视剧剧本转化为完整、可执行、可批量管理的场景图提示词。输出应像成熟影视美术部门的场景概念图需求，而不是剧情复述、人物海报或泛泛环境描述。

普通输出默认包含场景标题和连续机器提示词。每条提示词内部严格遵循 image-prompt-director 的视觉设计口径：先建立观看位置、空间层级与事件痕迹，再写材质、主光、颜色载体、介质响应与避免项。

## 总流程

1. 识别剧本文档结构：集数、场次、内外景、日夜、地点、场景描述、动作段落、台词提示。
2. 提取所有物理场景与重要场景状态。
3. 合并重复地点，保留会影响画面的变体，例如时间、天气、季节、破坏程度、陈设变化、仪式状态、战后状态、节庆状态、权力状态。
4. 剔除主要人物与角色表演，只保留场景本体和人物存在过的环境痕迹。
5. 为每个场景锁定一个视觉中心：建筑入口、街道轴线、室内陈设、桌案、屏风、院落、城门、营帐、码头、楼梯、窗光、道具痕迹或空间尺度。
6. 为每个场景确定摄影机位置、景别、空间层级、主光方向、颜色载体、成像介质与少量失败防护。
7. 输出可直接复制到生图工具的提示词。

## 场景粒度

“场景”不是每一次人物出场，而是一个可制作的空间资产或场景状态。

需要拆分为不同提示词的情况：

- 同一地点在日/夜、晴/雨/雪、清晨/傍晚有明显不同光线。
- 同一地点在平常、宴会、审判、战后、火灾后、废弃后、戒严后、节庆后有明显不同陈设或气氛。
- 同一场景需要正向视角、反向视角、鸟瞰、入口视角、室内核心视角等不同资产图。
- 剧情反复出现的核心场景需要“基础版”和“剧情状态版”。

需要合并的情况：

- 只是同一房间中人物位置不同，场景本体没有变化。
- 只是同一街道中的不同对话，没有新增陈设、天气或破坏状态。
- 剧本反复出现同一地点，但视觉状态完全一致。

## 场景信息提取字段

内部提取时关注以下信息，但最终不必全部展示：

- 集数与场次编号
- 原剧本地点名
- 场景统一命名
- 内/外景、日/夜、时间段
- 时代与地域
- 建筑类型、空间结构、入口出口、轴线
- 前景、中景、远景、背景
- 关键陈设、道具、地面、墙面、门窗、帘幕、桌案、旗帜、交通工具、摊位、器具
- 天气、季节、空气介质、烟、雾、尘、雨、雪、风
- 主光来源、方向、软硬、互动光
- 色彩载体：木、石、纸、布、漆、金属、雨地、雾、烛火、霓虹、屏风、旗帜等
- 场景功能：权力、生活、等待、逃亡、审判、战后、压迫、悬疑、温情、繁华、荒凉
- 必须排除的人物：主要角色、台词人物、正脸演员

## 提示词生成结构

每条提示词通常写成 2–4 个自然段：

第一段：场景身份、时代地域、场景状态、摄影机位置、观看方向、景别与构图。明确“无主要人物入画”或“空景”。

第二段：空间层级与可见美术细节。按前景、中景、远景组织，不平均堆砌所有物件；只写真正能定义场景的建筑、陈设、材质、地面、墙面、器物与事件痕迹。

第三段：主光来源、天气/空气、颜色载体、成像介质、清晰度层级。电影写实场景优先使用现场光、窗光、门洞光、烛火、阴天环境光、清晨斜光、雨夜反射等可信光源。

第四段可选：少数失败防护。避免主要人物、演员正脸、角色海报感、CG 塑料感、过度 HDR、满屏均匀高清、泛泛古风背景、无来源轮廓光、过量烟雾、现代违和物件等。

## 默认媒介口径

默认使用 `cinematic` 口径：画面像摄影机处在真实影视现场，场景承载事件前后留下的压力，而不是干净的游戏地图或旅游宣传图。

年代剧、古装、武侠、宫廷、战争场景：强调建筑轴线、门洞、廊柱、屏风、台阶、街市层次、旗帜、器物、地面痕迹与自然光，不做网游蓝金光和满屏粒子。

现代剧、商战、悬疑场景：强调真实地点表面、玻璃、办公室几何、走廊、便利店、车库、住宅、街道、雨地反射、实际灯和现场色偏，不做广告棚拍。

仙侠、玄幻、非现实场景：非现实元素只作为场景状态的一部分，受真实光、雾、遮挡、暗部、景深和画框裁切限制，避免全景 CG 资产展示。

## 参考图处理

如果用户提供参考图：

- 判断参考图承担的用途：建筑结构、材质、色彩、镜头视角、灯光、时代美术、空间气质或场景状态。
- 不写“参考图风格”作为提示词主体，而是把可见规律转译为具体空间、材质、光线和观看位置。
- 多张参考图合并时，只继承真正影响结果的元素，并统一光源、空间逻辑和时代美术。

## 导演模式

只有用户明确输入“导演模式”时，按以下字段输出，每个场景一组，不留空：

```text
【场景编号】【场景名称】【内外景/日夜】【场景状态】
【镜头角度】【镜头尺寸】【景别】【景深】【成像介质】
【场景主体】【核心空间结构】【主要材质/纹理】【事件痕迹】【人物控制】
【核心机位位置】【前景】【中景】【远景】【背景】
【主光源/方向】【光线软硬质感】【辅光源/互动光】【天气与空气介质】
【整体色温/基调】【色彩载体】【高光与阴影倾向】【对比度与饱和度】
【避免项】
```

## 自查

输出前确认：

- 是否已经提取所有可制作的物理场景与重要状态。
- 是否没有把人物资产、服装资产、道具展示误写成场景图。
- 是否默认剔除了主要人物。
- 是否每条提示词都有明确机位、空间层级、主光、颜色载体和电影介质。
- 是否避免了泛泛“古代街道/豪华宫殿/阴森房间”这类空词。
- 是否删除了无功能装饰、过量粒子、无来源补光和 CG 资产感。
