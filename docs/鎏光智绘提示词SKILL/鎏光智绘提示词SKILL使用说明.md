# 鎏光智绘 · 提示词 SKILL 使用说明

> 用这套 **8 个 skill** 把一部剧本做成"强控视频"的标准作业流程。
> 读者：导演 / 编剧 / 分镜师 / 美术师 / 后期。每个人只需看「我的角色」那几节 + 第 5 节三条硬约定。
> 核心链路：**影视化剧本 → 资产 → 静帧 → 视频 → 剪辑**。静态长相靠资产/参考图锁，动态戏靠提示词写。

---

## 0. 本套件目录（八个 skill，按中文文件夹名找）

```
鎏光智绘提示词SKILL/
  鎏光智绘提示词SKILL使用说明.md   ← 本文件
  剧本医生/          SKILL.md + references/ + scripts/
  人物主图/          SKILL.md
  服装设计卡/        SKILL.md
  道具展示图/        SKILL.md
  场景图/            SKILL.md + RUNTIME.md + references/
  群演选角板/        SKILL.md
  分镜大师/          SKILL.md + references/
  Seedance视频提示词/ SKILL.md + references/
```

- **文件夹名 = 大家平时叫的俗称**，skill 文内互相引用（如「人物主图」「场景图」「群演」）也用这套名，对得上。
- 每个 skill 入口文件固定叫 **`SKILL.md`**（Claude 靠它识别）；内部 `references/` 里的英文文件名是被 `SKILL.md` 用"见 references/xxx.md"指向的，**别改名**（改名要同步改所有指针）。
- 每个 `SKILL.md` 顶部 frontmatter 里的英文 `name:`（如 `script-revision`）是给 Claude 加载用的**技术 ID**，不影响你按中文文件夹找 skill，Kimi/Gemini 里也用不到。

## 3. 角色分工 × skill

- **导演（1 人）**：立项时定 **B/A/S 级**；把关剧本医生产出的资产清单；决定**静帧取舍**（哪些镜头值得出静帧）；统筹各线节奏。手上 skill：剧本医生（或指派编剧）。
- **编剧 / 改剧本（可由导演兼）**：跑**剧本医生**，产出影视化剧本 + 资产清单 + 环境音。这是全组的起点和命名权威。
- **美术师（2–3 人）**：按资产清单分工跑 5 个资产 skill（**并行**）。常见分法：美术 A＝人物 + 服装；美术 B＝场景 + 道具；美术 C＝群演（人少时合并）。产出各类资产母版图。
- **分镜师（1 人）**：先用**分镜大师**出静帧提示词→image2 出静帧（S/A 选择性）；再用 **Seedance** 出视频提示词→出强控视频。
- **后期（1–2 人）**：剪辑、调色、音效、配乐、拼接成片；必要时用 Seedance 的**首尾帧/局部编辑**修镜。

---

## 4. 标准顺序与并行（什么时候能同时干）

**串行骨干（不可乱序）**：剧本医生 → 资产 → 静帧 → 视频 → 剪辑。

**为什么是这个顺序**：资产/静帧/视频都**引用资产** → 资产先做；静帧是视频的**强控输入**（S/A）→ 静帧先于视频；一切从剧本派生 → 剧本最先。

**并行点**：① 资产阶段 5 条线**完全并行**（2–3 美术师同时开工，提速关键）；② 分镜师在资产期可**用剧本预规划镜头**，生成等资产；③ 后期可在第一批视频出来后**边出边剪**。

---

## 5. 三条跨阶段硬约定（让多线并行不散架）⚠️全组必读

**① 资产清单 = 唯一真源**
剧本医生产出的**资产清单**（每个 `@角色名 / @场景名 / @道具名` + 服装状态 + 环境音）是全组命名权威。美术按它命名资产、分镜师/Seedance 按它 @调用。**任何人不得私自改名或另起炉灶**；要改名，回剧本医生改清单、全组同步。

**② @资产名贯穿到底**
同一个名字从 **剧本 → 资产母版 → 静帧 → 视频** 全程不变。Seedance 的 `@杨延昭`、`@金殿`、`@断枪` 能对上，靠的就是这个名字一路没变。

**③ 分级（B/A/S）决定下游做法**
导演立项定级，写进项目头，传给分镜师：
- **S / A（精品）**：分镜大师走**影视级分镜块** + **选择性出静帧**；Seedance 用**分镜块骨架**。
- **B（跑量剧）**：静帧**可选/常跳过**；Seedance 用**八要素公式骨架**，效率优先。

---

## 6. 分级差异（B / A / S）

| 维度 | S / A（精品） | B（跑量剧） |
|---|---|---|
| 剧本医生 | 视听语言化 + 合规化 + 资产清单 | 同上（可更快） |
| 资产 | 该出的都出，质量拉满 | 关键资产为主，效率优先 |
| 静帧（分镜大师） | **选择性出**：复杂构图/多人/难控动作/关键情绪/会被反复参考的镜头 | **可选，常不做**；不做的镜头直接进 Seedance |
| 视频骨架（Seedance） | 影视级**分镜块**（逐镜强控） | **八要素公式**（快出量） |

## 7. 各 skill 速用卡

> 通用：默认引擎 **image2**；画幅开场问（横屏 16:9 / 竖屏 9:16 等）；默认**只做指定那一集**，不一次铺全剧；人物出镜句末加质量锚定语。

- **剧本医生**：输入剧本（.docx/.txt/.pdf 或粘贴）。先通读出诊断报告 → **等导演确认策略** → 再逐集改。两条准则：①视听语言（看不见听不见的外化或删）②合规化（血腥→暗红污渍、明星不 1:1、品牌虚构、规避反射镜头）。**多人/复杂动作/运镜/文字不在剧本阶段砍**（交给静帧强控）。产出：影视化剧本 + **资产清单** + 环境音。
- **人物主图**：只写**脸 + 毛发**，不写表情/服装/道具/场景。性格→骨相五官气质。统一正面证件照、3:4。参考图别拿明星照 1:1 还原。
- **服装设计卡**：**无脸假人模特**展示，只看戏服。只做主角/次要角色；群演服装看「群演选角板」。同一角色不同集的服装变化分别出。
- **道具展示图**：只展示器物。六分类 + S/A/B/C 重要度；S/A 单件 hero 档案，B/C 场景组合板（≤6–8 件）。
- **场景图**：**空景、无主要人物**。先用大模型把情绪简介扩成完整摄影语言再出；可给 HEX 锁色。
- **群演选角板**：真人、成组、**人脸高度差异化**；单板 6–10 人，多了拆板。它**同时是群演的服装参考**。
- **分镜大师**：从剧本（+资产）出关键帧静帧提示词。先**筛选/合并**（不是每句台词都出帧），再**静帧取舍**（S/A 选择性、B 可选）。输出 24 字段 + 可直接出图提示词。资产靠**参考图**喂 image2，不是首帧。
- **Seedance视频提示词**：把静帧/资产转成视频提示词。**多图参考（非首帧）**：静帧 + 资产母版用 @调用，静帧可选/选择性。一镜一运动、写力与物理、台词=一个镜头；单条 ≤15 秒、≤3000 字符、纯中文、禁配乐/字幕。

## 8. 跨工具使用（Claude / Kimi / Gemini）

**提示词制作首选 LLM 是 Claude**——这套 skill 就是按 Claude 的 **Anthropic Agent Skill** 格式（`SKILL.md` + frontmatter + `references/`）做的，在 Claude 上能力最完整（自动触发、自动按需加载 references、能跑脚本）。**没有 Claude 时再用 Kimi / Gemini 等。不同工具用法不同**：

- **Claude（推荐给分镜师/复杂线）**：当真正的 Agent Skill 用——放进 skills 目录，`description` 自动触发、`references/` 按需自动加载、`scripts/` 可跑（如剧本医生的 `extract_script.py` 自动分集）。
- **Kimi / Gemini / 其他**：当**提示词模板**用——把对应文件夹里的 `SKILL.md` 正文（**+ 当前需要的那一个 reference**）整段**粘进对话**，再贴上你的剧本/资产清单即可。注意：frontmatter 与"按需读 references"在这些工具里**不会自动生效**，所以要**手动**把需要的 reference 一起贴；剧本医生的 `extract_script.py` **只在能跑代码的环境用**，否则直接粘剧本文本让模型按"第X集"切。
- **统一前提**：无论哪个工具，**第 5 节三条硬约定照守**（同一套 @资产名、同一份资产清单、同一个分级）。

## 9. 文件交接与命名规范（项目工作目录）

> 注意：这是**做项目时的工作目录**，和上面第 0 节的"skill 套件目录"是两回事。

- **资产文件命名**：`@资产名_状态.png`，例：`杨延昭_定妆照.png`、`杨延昭_战损甲胄.png`、`金殿_日景.png`、`断枪.png`。`@资产名` 必须与资产清单一致（硬约定①②）。
- **建议工作目录**：
  ```
  项目名/
    00_剧本/ 影视化剧本.md  资产清单.md
    01_资产/ 人物/ 服装/ 道具/ 场景/ 群演/
    02_静帧/ 第X集/
    03_视频/ 第X集/
    04_成片/
  ```
- **交接清单**：剧本医生交「影视化剧本 + 资产清单（含 @名/环境音）+ 分级」；美术交「按 @名命名的资产母版图」；分镜师交「静帧 + 视频提示词 + 已用 @名」；后期交「成片」。

## 10. 一页速查

- **拿哪个 skill**：编剧→剧本医生；美术→人物主图/服装设计卡/道具展示图/场景图/群演选角板（并行）；分镜师→分镜大师→Seedance；后期→NLE。
- **三条硬约定**：① 资产清单是唯一真源 ② @资产名一路不变 ③ 分级定下游（S/A 分镜块+选择性静帧；B 八要素+静帧可选）。
- **顺序**：剧本 → 资产（并行）→ 静帧（S/A 选择性）→ 视频 → 剪辑。
- **通用**：image2、开场问画幅、默认只做指定集、句末质量锚定语、禁配乐/字幕/反射镜头、血腥→暗红污渍。
