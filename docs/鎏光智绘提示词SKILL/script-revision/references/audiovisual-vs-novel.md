# 视听语言 vs 小说逻辑：改写指南

核心命题：**小说写"读者在脑中想象什么"，剧本写"摄影机拍什么、话筒收什么、观众看到听到什么"。** 凡观众不能直接看见或听见的，必须转化为可拍的画面/动作/声音，或删除，旁白是下策。

下面按"小说式残留"的类型给改写方向和示范。示范用 → 表示"改成"。

---

## 1. 内心独白 / 心理活动

小说可以直接写"她心里想……"。镜头里没有"心里"。要外化。

外化的四种手法（优先级从高到低）：
1. **动作**：用一个可拍的具体行为体现心理。
2. **微表情 / 眼神 / 身体语言**：特写承载情绪（竖屏尤其依赖特写）。
3. **承载潜台词的台词**：让人物说一句"言在此意在彼"的话。
4. **环境 / 道具 / 声音**：用外部元素映射内心（下雨、停摆的钟、攥皱的照片）。

**示范**
- 小说："他强压怒火，心想这笔账早晚要算。" → 剧本：他端起茶杯，指节因用力而发白，慢慢吹开茶沫，淡淡一句："来日方长。"（动作+潜台词，"来日方长"是威胁的双关）
- 小说："她突然意识到自己被骗了，感到一阵天旋地转。" → 剧本：她手里的合同滑落，镜头随纸页飘落，再拉回她空洞的眼神。（动作+画面，不写"意识到"）

旁白只在以下情况用：第一人称强代入的赛道（如重生复仇开篇交代前世）、且确实没有更经济的外化方式时。即便用，也要短、要有态度，不复述画面已说的事。

---

## 2. 抽象定性（"他很善良""她城府很深"）

形容词不能拍。给一个**能拍的具体行为**让观众自己得出结论（show, don't tell）。

**示范**
- "他是个善良的人" → 他绕路把摔倒的盲人扶过马路，自己上班迟到被扣钱也没解释。
- "她城府极深" → 当众落泪博同情，转身回房，镜中的她面无表情，从容补好妆。

---

## 3. 背景交代 / 来龙去脉

小说可大段倒叙交代身世。剧本要把信息**溶进当下的冲突或画面**，不要让叙事停下来"上课"。

**示范**
- 小说花一段写"男主曾是战神，因被陷害而隐退" → 剧本：仇人当面羞辱他是废物，他不动声色；切一个极短闪回画面（战场上他一人退敌的背影）+ 老部下颤抖敬礼喊"将军"。信息在冲突中带出，还埋了反转钩子。

闪回要：短、画面化、有信息增量；不要用文字旁白复述。

---

## 4. 嗅觉 / 味觉 / 触觉等非视听感官

镜头收不到气味和味道。转成**视觉或听觉的替代**，或通过人物反应间接表现。

**示范**
- "厨房飘来红烧肉的香味" → 砂锅咕嘟冒泡、油光红亮的特写 + 男主闻香探头、喉结一动。
- "这汤难喝得要命" → 他喝一口，表情凝固，趁人不注意吐回碗里。

---

## 5. 大段环境 / 景物描写

提炼成 1-2 个**有信息量、可入镜**的视觉元素，服务于氛围或剧情，其余删。竖屏画幅小，不堆砌环境。

**示范**
- 三句写豪宅如何奢华 → 一个镜头：保镖林立的旋转楼梯 + 墙上巨幅家主画像（同时交代权势与人物关系）。

---

## 台词优化：去长、去假、去尬、找更好的说法

台词是微短剧的高频改区。逐条对症：

### 过长
观众没耐心听长台词，尤其竖屏。一句话能说清就不用两句；把信息拆到动作和画面里分担。
- 改法：删铺垫和重复，留最锋利的那半句。"我警告你，如果你再这样做下去，总有一天你会后悔的，到时候别怪我没提醒你" → "再走一步，你会后悔。"

### 假（不像人话 / 解释性台词）
人物为了让观众听懂而互相交代彼此都知道的事（"你可是我亲弟弟啊"），或说书面腔。
- 改法：删掉"互相科普"，靠情境和称呼自然带出关系；把书面语换成口语。

### 尬（用力过猛 / 不合身份 / 时机不对）
强行煽情、强行幽默、与人物身份口吻不符。
- 改法：让情绪"少说一点"，靠潜台词和留白；强者往往话少，狠话冷说比喊出来更有压迫感。

### 更好的说法（升级，不只是修补）
在通顺的基础上追求"更妙"：
- **一语双关 / 潜台词**：表面一层意思，底下一层意思，呼应处境或后文。
- **留余地**：不把话说满，"来日方长""走着瞧"比直接威胁更有戏。
- **反差**：温柔的话配狠厉的事，或平静语气说惊人之语，制造张力。
- **人物专属口头禅 / 说话节奏**：强化辨识度。

台词改写黄金问题：**这句话，这个人在这个处境下，真的会这么说吗？有没有更短、更像他、更有潜台词的说法？**

---

## 一个统一的自检动作

改任何一处时问：
1. 观众能"看见/听见"这个吗？看不见就外化或删。
2. 这个信息能不能用画面或动作代替台词来传递？能就别用嘴说。
3. 删掉它，戏会塌吗？不会塌就是闲笔，删。
