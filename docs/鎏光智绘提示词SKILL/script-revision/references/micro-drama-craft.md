# 竖屏微短剧创作法

微短剧靠**完播率**和**付费转化**两个指标活着。所有节奏判断都回到一个问题：**观众会不会划走？会不会愿意为下一集付费？** 这套逻辑和长剧的"娓娓道来"完全相反。

---

## 钩子（Hook）：把人勾进来、钉住

钩子分两种位置，每集都要有：

**集首钩子**——开头 3 秒内必须给一个让人想看下去的东西：一句炸裂的台词、一个高冲突画面、一个未解的悬念。**第一集的集首钩子决定全剧生死**，要把最大的冲突点、反转点、或最强的爽/虐点往最前放（哪怕打乱时间线，用冷开场+倒叙）。

**集尾钩子（卡点/勾子）**——结尾必须留一个悬念或反转，让观众非看下一集不可。常见卡点：
- 真相即将揭晓的前一秒切断（"其实我就是……"——黑屏）。
- 危机突降（门被推开、电话响起、反派现身）。
- 反转预告（前一秒得意，镜头给出观众才看到的危机）。
- 身份/马甲即将掉落的临界点。

诊断时逐集核对：**每集开头有没有钩子？结尾有没有卡点？** 没有就是掉人风险点。

---

## 反转（Reversal）：微短剧的发动机

反转 = 观众的预期被打破。密度要高（常见每 1-3 集一个中小反转，关键节点大反转）。常见类型：
- **打脸反转**：被看轻者亮出真实身份/实力，羞辱者当场难堪（爽点核心）。
- **身份反转 / 马甲掉落**：隐藏的真实身份揭露（扮猪吃虎、马甲文命脉）。
- **关系反转**：敌变友、友变敌、亲人是仇人。
- **信息反转**：观众和角色发现之前的认知是错的。
- **攻守反转**：被动一方突然掌握主动。

改剧本时，若某段平淡，常见解法是**补一个反转或把已有反转前置/加强**。但反转要有铺垫和合理性，不能为反转而反转（突兀的反转观众会觉得假）。

---

## 爽点与虐点

**爽点**——让观众情绪释放、解气：打脸逆袭、扮猪吃虎、一鸣惊人、扮演者被打脸、隐藏大佬身份曝光、绝境翻盘。爽点要"先抑后扬"：先压制（被欺负、被看轻），再爆发，落差越大越爽。

**虐点**——让观众揪心、共情：误会、错过、牺牲、求而不得、亲人离散。虐点制造情感投入，但微短剧的虐通常"虐得快、解得也快"，长时间纯虐会掉人。

诊断爽/虐设计：是否够密、够准、够狠？落差够不够大？有没有"压制得太久才爆发"导致前段掉人？

---

## 付费点设计

平台通常在某一集设付费卡点（免费看前 N 集，之后付费）。两件事最关键：
- **卡点那一集**：悬念/反转必须拉到最满，让观众"不付费睡不着"。
- **卡点前的免费集**：节奏要持续加速、把人勾到欲罢不能，绝不能在卡点前出现疲软或慢热。

诊断时若已知卡点位置，重点评估卡点集强度和前置免费集的"勾人曲线"是否一路向上。

---

## 进阶手法在短剧里怎么用（克制而精准）

用户要的不止套路，还要这些更高级的处理。短剧时长压缩，它们要用在刀刃上：

- **开放式取景 / 冷开场**：不交代来龙去脉，直接把观众丢进一个高信息/高冲突的画面，制造悬念再补因果。做集首钩子极有效。例：开篇直接是男主跪在灵堂、满室缟素——是谁死了？为什么跪？观众被勾住。
- **平行 / 交叉叙事**：两条线交替推进，在一条线的悬念顶点切到另一条，既制造钩子又压缩时间。短剧里用于卡点（在高潮处切走）和制造"观众知道、角色不知道"的张力。注意别切太碎，竖屏观众跟线能力有限。
- **留白**：不是放空，是**反转或情绪爆发前的半秒静默/停顿**，用以放大冲击；或一个意味深长的眼神不配台词，留给观众回味。短剧的留白要短、要准，长留白=掉人。
- **情绪烘托**：靠音乐起范、关键特写、环境光影一两笔带出情绪，而非冗长铺垫。一个雨夜、一段配乐、一个颤抖的手部特写，比三句内心独白更有效也更省。

这些是"必要时"用，不是堆砌。每用一次问：它服务了钩子/情绪/悬念吗？还是只是炫技拖了节奏？

---

## 分赛道套路速查（识别赛道、贴合期待）

通读时先判断赛道，不同赛道观众期待不同，改时要贴合：

- **战神 / 兵王 / 赘婿归来**：被轻视的强者隐藏身份→受辱隐忍→关键时刻亮出战神/财阀身份打脸。爽点密集，身份反差是核心。
- **霸总 / 豪门**：先婚后爱、契约、误会、追妻火葬场。强情绪拉扯，男主"又狠又宠"。
- **重生 / 穿越 / 复仇**：带着记忆/金手指重来，复仇打脸、改写命运。开篇常用旁白交代前世，要快。
- **甜宠**：高糖、强互动、轻冲突。钩子靠"甜的预期"和小误会。
- **古装宅斗 / 宫斗**：身份、算计、反转。注意场景实拍成本（见 filmability）。
- **逆袭 / 扮猪吃虎 / 马甲**：隐藏实力或身份，层层揭马甲，每掉一个马甲一个爽点。

判断赛道后，诊断"是否给足了该赛道观众想要的核心爽感"，改动时强化赛道命脉、删与赛道无关的旁枝。
