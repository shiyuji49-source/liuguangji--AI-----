# 实拍可行性：判断清单与取舍示范

核心原则：**戏剧效果第一，呈现成本第二。** 凡是高成本/难拍的元素，先问"它推动剧情吗？是这场戏的核心吗？"——
- **是核心** → 保留，转而想低成本的拍法。
- **只是氛围/装饰** → 换成同等效果、更好拍的版本，或删。

竖屏微短剧普遍低成本、快周期（全剧常 7-15 天拍完、单集成本极低），所以这一步不是优化，是"能不能拍出来"。但**绝不为省钱牺牲爽点、反转、钩子**——那是微短剧的命。冲突时：留住戏剧效果，简化手段。

---

## 高成本 / 难拍元素红旗清单

通读时遇到下列元素，标记位置，逐个做取舍判断：

| 红旗 | 为什么贵/难 | 常见简化方向 |
|---|---|---|
| 特殊天气/天象（血雨、陨石、极光、龙卷风、海啸） | 特效或难等天时，布景反复清理 | 不影响剧情就换普通天气（血雨→暴雨；陨石→流星一闪）；是核心则用局部特效/声音+反应戏暗示 |
| 大规模 CGI / 奇幻特效（飞天遁地、法术、巨兽） | 后期成本高、易穿帮 | 用"以小见大"：特写局部+音效+人物反应，避开全景奇观；或改为发生在画外、只拍结果 |
| 爆破 / 火灾 / 车祸等高危特技 | 安全成本、需专业团队 | 拍"前因+反应+后果"，省略过程本身；或用声音和光影暗示 |
| 大量群演 | 人力与调度成本 | 竖屏窄画幅本就装不下人群——用前景几人+虚化背景暗示"很多人"；改为小范围冲突 |
| 动物 / 婴幼儿出镜 | 不可控、超时、需专人 | 减少其镜头量；用画外音、局部特写、道具替代（如婴儿哭声+襁褓特写） |
| 稀有/难搭场景（古战场、皇宫大殿、异世界） | 搭景或借景昂贵 | 集中在小而精的角落取景；一堂景多用；改写到更易得的场景（御书房代替大殿） |
| 频繁转换地点 | 转场即转码头，烧时间 | 合并场次到同一/相邻场景；把分散的戏攒到一堂景里发生 |
| 同一日内大量日/夜切换 | 打乱拍摄排期、需补光或等光 | 合并时间，减少日夜来回切换，方便集中拍摄 |
| 大量服化道更换 / 年代跨度大 | 成本与连戏管理 | 精简造型变化；跨度大的部分用关键道具/字幕点明，不靠满屏置景 |
| 水下 / 高空 / 极端环境 | 专业设备与安全 | 改到可控环境，或用近景+局部+声音营造 |

## 取舍示范（→ 表示改成）

- 天降血雨（仅渲染压抑氛围，不推动剧情） → 暴雨倾盆。理由：血雨需调色/特效/清洗布景，暴雨一台雨车即可，氛围等同。
- 血雨是"预言成真"的剧情关键 → 保留，但只拍：雨滴砸在白衣上晕开红痕的特写 + 众人惊恐仰望的反应 + 一声惊雷。局部呈现，避开全景大特效。
- 男主一掌劈山（玄幻爽点核心） → 保留冲击力，改拍法：蓄力特写→画面剧烈震动+碎石飞溅近景→尘埃落定后的山体裂缝远景一闪+对手骇然反应。避开"完整劈山过程"的大特效。
- 千军万马攻城 → 城头几名将领的近景+箭雨入画+喊杀声（画外）+一两个冲锋的前景士兵虚化背景。以小见大。
- 主角开豪车飞驰撞向反派（车祸） → 反派惊恐回头特写→刺眼车灯逼近+刹车尖啸（声音）→黑场→现场狼藉的后果镜头。省去碰撞过程。
- 一场戏横跨"白天的市集→黄昏的码头→深夜的宅院" → 若剧情允许，压缩到傍晚的同一片区域，减少日夜与转场。

## 给诊断报告的"实拍总评"怎么写

- 统计全剧红旗数量与分布（哪几集集中）。
- 区分"可直接简化"（不影响剧情）与"需保留但要优化拍法"。
- 估一个粗略的成本风险等级（高/中/低），帮制片判断。
- 不替制片做预算决定，只把可拍性问题和可选方案摊清楚。
