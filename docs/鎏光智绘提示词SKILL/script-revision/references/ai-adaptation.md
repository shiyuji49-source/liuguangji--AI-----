# AI 生成适配（合规 · 资产建档 · 生成硬伤规避）

> 本文件服务"影视化剧本优化（分镜前置）"。流水线是 image2 出资产/静帧、Seedance 出视频，**静帧强控**承担了大部分"画面能不能成"的问题。所以剧本阶段只盯**静帧也救不了**的几样，外加产出**资产档案**喂下游。

---

## 一、内容合规转换对照表（剧本阶段就改干净）

合规属于**平台/审核**问题，image2 和 Seedance 两端都会拦，静帧救不了。在剧本里就转换：

| 原始描述 | 改成 |
|---|---|
| 血腥、喷血、断肢、尸体特写、伤口 | 战损痕迹、**暗红污渍**、破损衣甲、染尘、不直视伤口的**侧面构图**、远景模糊倒影、以光影暗示创伤 |
| 露骨/色情/过度性暗示 | 删，或转含蓄（眼神、气氛、留白） |
| 酷刑、极端暴力直白展示 | 压抑氛围 + 环境暗示（工具空镜、阴影、声音），不直给过程 |
| 真实明星/公众人物长相 | 虚构面孔，只借"气质方向"，不 1:1 还原（肖像合规） |
| 真实品牌/商标/logo | 虚构品牌名与标识 |
| 危险/违法行为的可操作演示 | 淡化、模糊处理，不给"教程式"细节 |

> 原则：**留住戏剧效果，换掉会被拦或显假的呈现方式。**（与道具/服装/分镜大师/Seedance 手册口径一致。）

## 二、生成硬伤规避（静帧也救不了的极少数）

- **反射镜头**：别让关键叙事信息只靠镜子/水洼/刀面/玻璃的**反射**呈现。Seedance 一动就破坏反射几何（脸崩、字乱、空间错）。
  - ❌ 刺客的脸只在刀刃反光里出现；凶手身份靠镜中倒影揭示。
  - ✅ 让对象**直接入画**：侧身半张脸进光、回头、从遮挡后转出。
- **同一连续镜头里出框又回来**：Seedance 把"出框"当成隐性剪辑。若剧情需要人物离开再回来，剧本里就当作**两个镜头/两场**写，别写成一个长镜头里来回。
- **极端矛盾的同时要求**：如"超高速运动 + 画面极度稳定"。剧本里别把互斥的视觉要求绑在同一瞬间。

---

## 三、哪些**不必**在剧本阶段回避（交给静帧强控）

下面这些常被误以为"AI 做不到"，但本流程用 image2 静帧 + 首尾帧锚定就能控住，**剧本照常写**：

- **多人同框 / 群像场面**：静帧把人物站位、人数、面孔先定死，Seedance 再动。（注意：单条 Seedance 提示词跨剪辑稳定追踪 ≤3 人是**下游分镜/视频提示词**的事，不是剧本要砍人。）
- **复杂动作 / 打斗**：用关键瞬间的静帧 +（必要时）首尾帧锚定来控；很复杂时下游会拆更多镜头。
- **高速 / 花式运镜**：运镜规范由下游 Seedance 提示词把控，剧本只管"这里要强烈的视觉冲击"。
- **画面文字 / 招牌 / 字幕信息**：可在静帧阶段渲染或合成，剧本可以保留有文字的画面。

## 四、资产清单建档格式（喂资产提示词 skill；保证后续 @ 调用一致）

剧本阶段为复用对象建档，全程同一段描述不漂移：

**固定角色档**（每个角色一条）：
```
@角色名：年龄 + 身份 + 发型 + 妆容 + 服装 + 眼神气质。全程固定五官、发型、穿搭不变。
（如有多形态：素衣态 / 黑化态，分别建档，并注明"两态同一张脸"。）
```

**固定场景档**（每个空间一条）：
```
@场景名：空间结构 + 背景陈设 + 光线方向/性质 + 环境音。
环境音示例：古装殿内→"烛火爆裂声、远处更夫锣声"；现代办公→"空调低鸣、键盘敲击"。
```

**道具 / 服装清单**：
```
道具：@道具名 = 形制 + 材质 + 颜色 + 使用方式（剧情关键道具才单列）。
服装：@角色名 在第X集/某场 的服装状态（常服/官服/战损/夜行…），变化要逐集标。
```

> 配比参考（Seedance 手册）：出视频时形象参照 1–2 张 + 环境参照 1 张 + 动态参照 1 段 + 声场参照 1 段，共 4–5 个，不贪多。剧本的资产清单就是这些参照的来源台账。

---

## 五、与下游对接时要记得的 Seedance 硬约束（剧本阶段顺带留意）

这些主要在写 Seedance 提示词时生效，但剧本结构上提前照顾会更顺：

- **单镜几秒、单条 ≤15 秒**：剧情节拍尽量能落到"若干秒一个清晰画面"，便于下游切镜。
- **一句台词 = 一个镜头**：关键对白尽量是"一人一句、对谁说清楚"，便于下游给每句配特写。
- **跨剪辑追踪 ≤3 个角色**：一段连续戏里真正需要"全程跟住"的核心人物别超过 3 个（群众可在背景，不必逐一追踪）。
- **环境音优先、配乐后期加**：剧本标环境音即可；配乐不写进画面提示。

---

## 一句话

剧本医生在 AI 流程里只做三件事：**视听语言化**（看得见听得见）、**合规化**（血腥/露骨/明星/品牌/反射转换）、**建资产清单**（角色/场景/道具/服装 + 环境音）。能力性的"画面难不难"留给静帧强控，别在剧本里自我设限。
