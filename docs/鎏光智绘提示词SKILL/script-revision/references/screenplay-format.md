# 竖屏微短剧剧本格式规范

输出修改后剧本时用规范格式，方便顺场、拍摄和制片统筹。下面是行业通用的简洁格式（微短剧不必像院线剧本那么繁复，但要素要全）。

## 集与场的层级

```
第1集

场1  内  顾家别墅·客厅  日
【人物】顾廷川、苏晚、管家
（动作提示／舞台说明顶格或用 △ 引导）

苏晚（冷笑）
台词内容写在人物名下方。

顾廷川
台词内容……（括号内是表演提示，简短）

——本集卡点提示（可选）：写明本集结尾的悬念/钩子是什么
```

## 各要素说明

- **集标题**：`第X集`，独占一行。
- **场景标题（场标）**：`场号 + 内/外 + 地点 + 日/夜`。
  - 场号：场1、场2……（便于顺场和通告）。
  - 内/外：决定打光与场地。
  - 地点：具体到可执行的场景（"顾家别墅·书房"而非"某处"）。
  - 时间：日/夜/晨/黄昏（影响排期，尽量减少同场内日夜来回切换）。
- **人物提示**：本场出场人物，方便统筹演员通告（可选但推荐）。
- **动作 / 舞台说明**：描述可拍的动作、表情、走位、关键道具、镜头提示。用 △ 或直接成段。**只写镜头能拍到的**，不写内心活动（内心要外化，见 audiovisual-vs-novel.md）。
- **对白**：人物名单独一行，台词在其下方；表演/语气提示放人物名后的括号里，要短（如"冷笑""压低声音"）。
- **景别 / 镜头提示**（按需）：竖屏以近景、特写为主，需要时在动作行注明（如"特写：苏晚攥皱的照片"）。不必逐镜分切，给到关键镜头意图即可。
- **卡点提示**（可选但对微短剧很有用）：每集结尾标注本集悬念/钩子，方便核对完播设计。

## 竖屏构图注意

- 9:16 画幅，**优先近景/特写、一到两人入画**；避免写需要横向大场面调度的画面。
- 群体/宏大场面用"前景几人 + 背景虚化/画外"暗示，别写满屏人海。
- 关键信息（道具、表情）适合用特写承载，写动作提示时可主动指出。

## 输出文件形式

- 默认输出 markdown（`.md`），便于在对话里预览和快速迭代。
- 用户需要成片/交付用的 Word 文档时，调用 docx skill 排版成 `.docx`（统一字体、场标加粗、对白缩进等）。
- 保持与原剧本一致的集号、人物名，不要擅自改名或改集数结构。
