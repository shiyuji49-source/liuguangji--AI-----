# Seedance 2.0 视频提示词生成器

拿到【分镜大师的关键帧/静帧】+【已做好的资产母版】，产出可直接复制进 Seedance 2.0 的中文视频提示词。**只产提示词，不生成视频，不调用生成工具。**

## 在工作流里的位置

```
剧本医生（影视化/分镜前置）
  → 资产提示词（人物/群演/服装/道具/场景 skill）→ image2 → 资产母版
  → 分镜大师（静帧提示词）→ image2 + 资产 → 静帧
  → 【本 skill】→ Seedance 2.0 + 资产 + 静帧 → 强控视频 → 剪辑（调色/音效/配乐）
```

## 强控来源：多图参考（不是首帧逻辑）⚠️核心

静帧和资产母版**都作为参考图**喂 Seedance，用 @ 标记调用（如 image1=静帧/构图参考、image2=主角、image3=场景、image4=道具……，handle 顺序固定）。**这不是"图生视频首帧"——静帧不是被锁死的第 1 帧，只是多图参考里的"构图/光影/blocking 锚"。** 由此三条贯穿全程：

1. 提示词**仍要写空间布局**（⚠️布局块：米数/朝向/谁遮谁/视线锁定），强化一致性，**不能指望首帧锁住**（写法见 `references/ASSET_SPACE_EDIT.md`）。
2. 笔墨**重点放静帧给不了的动态**：运镜、表演、物理、切点、台词、声音。
3. 静帧/资产图已带的纯静态外观细节**不必逐字复刻**（参考图已锁），省下 3000 字符额度给动态。

> **静帧是可选 / 选择性参考**：不是每条视频都喂静帧。S/A 精品只对**需要强控的镜头**放静帧参考；**B 级跑量剧静帧可选、不固定**——很多镜头无静帧，直接用资产母版做多图参考 + 八要素骨架。有静帧就当强构图锚，没静帧就靠资产参考 + 提示词写清空间布局。

## 开场询问（最多一次）

1. **画幅比例**：横屏 16:9 / 竖屏 9:16 / 院线 21:9 等。
2. **项目等级**：S / A / B（B = 跑量剧）。

两项已知就不再问。

## 等级 → 骨架（硬规则，**不做单镜覆盖**）

- **S / A 级（精品）→ 影视级分镜块骨架**：逐镜强控（@资产声明 / ⚠️空间布局 / 逐镜机位·运动·背景·动作①②③·微表演 / 风格块 / 禁清单）。
- **B 级（跑量剧）→ 八要素公式骨架**：主体+动作+场景+光影+镜头语言+风格+画质+约束，快出量。

按等级**一锁到底**，单条镜头不另切骨架。两套骨架完整写法与成品示例见 `references/SKELETONS.md`。

## 底层原则（两套骨架共用 · 9 条共识）

1. 先锁资产再动画（静态长相靠资产/参考图，动态戏靠提示词）。
2. **一镜一运动**：推/拉/摇/移/环绕，一段只用一个；先定景别再定唯一运镜。
3. 物理化表演：情绪词打底 + 关键处补 1-2 个可观测物理细节（喉结吞咽、下颌微抖）。
4. **写力与后果，不写字幕式动作**：✅"一拳→后坐力掀发、脚稳握杆"；❌"机甲打了怪物"。
5. 把摄影机当现场的人：静止→突然一震、急甩、推进、绕拍、撞击抖动、手持呼吸感。
6. 人物出镜**句末加质量锚定语**（见下）。
7. **干净输出**：只留环境音，**禁配乐、禁字幕、禁水印**（配乐留剪辑加）。
8. **时长**：单条 ≤15 秒、≤3000 字符；单镜通常几秒（闪现镜 0.3-0.5s）。
9. **引擎硬约束**：跨剪辑稳定追踪 ≤3 个角色；出框 = 隐性剪辑（不在同一连续镜头里出框又回来）；只描述能看见/听见的；**禁反射镜头**（刀面/水洼/镜子会让 Seedance 破坏几何）。

## 时间码 or 连续镜头

- **要内部快剪** → 时间码（如 0-5s/5-10s/10-15s，每镜五要素：运镜/构图/动作/对口型台词/光影，情绪几何递增）。
- **要一镜到底** → **不写**秒级时间码，整体写"一个连续镜头，不要分段或定义节拍"（写死每秒节拍会逼 Seedance 乱切）。
- 法则：**想切就用时间码，不想切就写连续镜头。**

## 质量锚定语（人物出镜句末必加，复制即用）

> 面部稳定不变形、五官清晰、人体结构正常、动作自然流畅、不僵硬、画面无卡顿、无闪烁。

## 输出与自查

纯中文（镜头术语中英对照见 `references/GENRE_PITFALLS.md` 附录）；台词写引号内、**保留原文不翻译**、一句台词=一个镜头、写明对谁说。输出前静默确认：@资产/handle 声明齐 → 空间布局清（米数/朝向/遮挡/视线）→ 一镜一运动 → 运镜绑情绪 → 物理化表演 → 该放的禁清单已放 → 质量锚定语已加 → ≤15 秒/≤3000 字符 → 纯中文、避开反 AI 词。
