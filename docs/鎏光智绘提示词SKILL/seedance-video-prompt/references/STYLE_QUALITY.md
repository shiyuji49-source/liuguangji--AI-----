# 风格 · 灯光 · 画质 · 约束

## 一、灯光哲学（全部题材统一写实派）

**所有等级（S/A/B）一律写实派**：⚠️**只用实景光（practicals）**——屏幕/窗户/台灯/烛火等场景内实际存在的光源；**禁一切电影补光**（正面光/侧补光/顶底光/反光板/柔光箱/LED 灯带/霓虹/画外光源）；摄影机始终在**阴影侧**拍摄，practicals 环境光照亮脸部轮廓与眼神光（catchlight）；全程大气薄雾 haze，**禁可见光束（god rays）**。不做风格派炫光分支。

## 二、配色 60:30:10（主色/辅色/点缀，按情绪选）

| 情绪 | 比例 |
|---|---|
| 冷峻冲突 | 60% 冷蓝 + 30% 混凝土灰 + 10% 红点缀 |
| 温暖亲密 | 60% 暖钨丝 + 30% 深棕 + 10% 琥珀 |
| 危险恐慌 | 60% 深红 + 30% 黑 + 10% 钠橙 |
| 科幻科技 | 60% 黑 + 30% 冷灰 + 10% 紫/青点缀 |

## 三、画质词

4K/超高清/细节丰富/锐度清晰；电影质感/IMAX 胶片/Panavision C 系列(35mm,f4)/边缘柔焦/胶片颗粒；无穿模/无重影/无闪烁/画面稳定；60fps 流畅、180° 快门运动模糊。

## 四、质量锚定语（人物出镜句末必加，复制即用）

> 面部稳定不变形、五官清晰、人体结构正常、动作自然流畅、不僵硬、画面无卡顿、无闪烁。

## 五、标准「禁」清单（按需丢进提示词）

```
禁3D渲染、禁游戏引擎/游戏CG过场质感。禁正面光/侧面补光/反光板/柔光箱/LED灯带/霓虹。
禁屏幕蓝光溢出。禁可见光束(god rays)。禁光学畸变/桶形畸变/鱼眼/广角变形。
禁漂浮道具。禁身份漂移。禁抖动(除手持呼吸感)。禁音乐。禁字幕。禁过饱和/禁HDR感。
禁反射镜头(刀面/水洼/镜子,会破坏几何)。
```

## 六、失效模式防御

重要用单 `⚠️`、关键用 `⚠️⚠️⚠️`，标注：任何距离、物体位置、各种「禁止」、关键站位、视线锁定、台词/动作时机、规则的例外。规则冲突时写「⚠️替换规则：原[X]替换为[Y]」，**不要默删**。
