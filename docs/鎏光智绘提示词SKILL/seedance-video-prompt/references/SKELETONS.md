# 骨架模板（八要素 / 影视级分镜块）

两套骨架按**项目等级**选定（S/A=分镜块、B=八要素），一锁到底，不做单镜覆盖。底层原则一致（见 SKILL.md 9 条共识），只是颗粒度不同。

---

## 一、八要素公式骨架（B 级跑量剧）

> **主体 + 动作 + 场景 + 光影 + 镜头语言 + 风格 + 画质 + 约束**

| 模块 | 写什么 | 示例 |
|---|---|---|
| 主体 | @角色（参考图锚点，保持稳定） | 一个男人（@image2，保持资产脸型/穿搭） |
| 动作 | 分段写、10 秒内 | 撕裂腰带 → 生化增殖 → 痛苦嘶吼 → 变身完成 |
| 场景 | @场景 + 时间天气 | 末日城市天台，战火，灰蓝阴天 |
| 光影 | 主调 + 关键光效 | 低饱和灰蓝，紫色光从核心炸裂，暗部压抑 |
| 镜头语言 | 一镜一运动 + 手持感 | 一镜到底，仰拍左侧→急速环绕至正面→右侧俯拍，手持呼吸感 |
| 风格 | 题材美学 | 暗黑特摄、生化机甲、战损痛苦美学 |
| 画质 | 质感词 | IMAX 胶片、Panavision、4K、颗粒、边缘柔焦 |
| 约束 | 质量锚定语 | 面部稳定不变形、人体结构正常、动作流畅不僵硬、无闪烁 |

**开头可固定一个「画幅与质量标准块」**（按题材替换风格词）：
```
竖屏 9:16（横屏 16:9 / 院线 21:9），总时长 15 秒整，8K 超高清，60 帧高帧率，写实微电影质感，
杜比视界 HDR，广色域，<风格色调>，真实服装织物纹理（丝绸反光、刺绣细节拉满），
光影层次丰富，无穿模、无畸变、无 AI 失真，口型与台词 100% 精准同步，情绪、微动作与台词语境完全匹配。
```

**快剪时的时间码五要素法**（B 级常用）：15 秒 = 约 5 次运镜、每 3 秒一镜，每镜写：运镜方式 / 画面构图 / 人物动作 / 对口型台词 / 光影+运镜细节，情绪曲线几何递增。

## 二、影视级分镜块骨架（S/A 级精品）

一条提示词按固定顺序写满：

```
1. @资产/handle 声明：@image1=静帧/构图参考；@image2=主角（外观、服装、状态如"湿发贴额"逐项写死）；@image3=场景；@image4=道具……
2. 通用警告：⚠️空间布局 / ⚠️对白规则（一句台词=一个镜头）/ ⚠️本视频严格只有 N 个镜头
3.【镜头1】机位（焦段+光圈+景别+手持/static/dolly）/ 摄影机运动（绑情绪）/ 背景 / 动作（①②③节拍）/ 微表演细节
  【镜头2】（紧接硬切）……
  —— 单镜则写「单镜头(one-shot，无剪辑)」，不分【镜头N】
4. 风格块（灯光哲学+配色+画质，见 STYLE_QUALITY）
5. 环境活动（背景在干什么，禁空旷背景）
6. 收尾：15秒。<画幅>。
```

每个镜头块固定含：**机位 · 摄影机运动 · 背景 · 动作 · 微表演细节**。

另一种更轻的连续字符串分段（标签内联，适合简单单镜）：
> **Style & Mood（风格氛围，必写）/ Narrative Summary（一句剧情，可删）/ Dynamic Description（逐镜动态）/ Static Description（场景道具）/ Audio（仅对白场景：台词+音效）**

> 注意：多图参考下，Static Description 不必复刻静帧/资产图已锁的外观，只写强化空间关系所需的最少静态信息（见 ASSET_SPACE_EDIT）。

---

## 三、时间码 or 连续镜头（决策）

- 想要内部快剪 → 用时间码（每镜五要素，情绪几何递增）。
- 想要一镜到底 → **不写**秒级时间码，整体写"一个连续镜头，不要分段或定义节拍"。
- 法则：**想切就时间码，不想切就连续镜头。**（写死每秒节拍会逼 Seedance 乱切。）

## 四、成品示例

### 八要素示例（B 级 · 暗黑特摄变身）
```
一个男人（@image2，保持资产脸型/穿搭）在末日城市天台，战火背景，灰蓝调阴天。
他猛然撕裂腰带，引发肉体生化增殖：甲壳刺破皮肤、彩虹碎片重组、乳白肌肉覆盖、胸甲闭合、
左胸盘炸裂展开半透明虫翼（布满血管）。仰天长啸，面部被生物薄膜吞噬拼出昆虫面具，复眼亮起淡紫光。
虫翼高频震动，垂直弹射起飞，空气撕裂，留生物磷粉。
光影：低饱和灰蓝，紫色光效从核心炸裂。一镜到底：左侧仰拍→急速环绕至正面→右侧俯拍，手持呼吸感。
暗黑特摄风格，IMAX 胶片质感，4K 超高清。10 秒内完成变身。
面部稳定不变形、人体结构正常、动作流畅自然、画面无闪烁。9:16。
```

### 分镜块示例（S/A 级 · 院线古装，配静帧"金殿对峙帧"做构图参考）
```
@image1=静帧"金殿对峙帧"（构图/光影参考）；@image2=杨延昭（脸型/发型/气质）；@image3=金殿；@image4=断枪。
⚠️空间布局（主视=屏风侧前方俯视）：杨延昭立殿心、面朝屏风；屏风后人影距其约 4 米、在其正前被屏风部分遮挡。
单镜头(one-shot，无剪辑)。
机位：85mm F1.4，屏风侧前方低角度 static 起幅，随他抬眼轻微一震、缓慢推进到面部中近景（总位移≤15cm）。
动作：①肩背端正→②指节因隐忍微微收紧→③目光直刺屏风后人影。
微表演：喉结一次吞咽，下颌微抖一次立刻收紧（情绪=压抑 + 物理细节）；高潮不眨眼。
风格：8K 写实，冷调青石灰 60% + 暗木 30% + 断枪暗红绳 10%；仅实景光（殿门高侧硬光自左上）、摄影机在阴影侧、全程薄雾、禁可见光束。
环境活动：烛火轻晃、远处更夫锣声。
禁海报站姿、禁均匀轮廓光、禁网游仙侠发光、禁反射镜头、禁过度血腥、禁字幕、禁音乐。
面部稳定不变形、五官清晰、人体结构正常、动作自然流畅、不僵硬、画面无卡顿、无闪烁。
（可选台词）"你也配问我？"——带克制怒意的低声。16:9。15秒。
```
