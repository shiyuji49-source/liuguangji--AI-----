# 多图参考 · 空间走位 · 延展组接

## 一、多图参考机制（不是首帧逻辑）⚠️

静帧和资产母版**都是参考图**，用 @ 标记调用，Seedance 据此对齐外观与构图。**静帧不是被锁死的第 1 帧**，只是多图参考里的"构图/光影/blocking 锚"——所以仍要靠提示词写清空间关系来强化（见下方空间走位块）。

- **handle 固定 + 顺序很重要**：固定编号（如 image1=静帧/构图参考、image2=主角、image3=场景、image4=道具…），描述里谁先出现就先放谁；用分镜表后顺序自动保持，无需每次重传重标。
- handle 每条提示词可重新编号（同一 @image1 在不同镜头可指不同资产，需当条声明）。
- 同一人物多形态（素衣态/黑化态）：用「多模态参考图锚点，确保两张脸完全一致」。
- 因为是多图参考：**静帧/资产图已锁的纯静态外观不必逐字复刻**，省下 3000 字符给动态；但凡是模型易错的**空间关系**仍要明写。

### 四类参考素材配比（不贪多）
形象参照 1-2 张 + 环境参照 1 张 + 动态参照 1 段 + 声场参照 1 段（共 4-5 个）。

| 类型 | 作用 | 要点 |
|---|---|---|
| 形象参照 | 锁角色外观/服装/气质 | 1-2 张就够，别给同一人冲突照片 |
| 环境参照 | 固定空间气质/光线/材质 | 是「场景稳定器」，不是装饰 |
| 动态参照 | 提供镜头轨迹与动作节奏 | 学**运动轨迹**，非复刻原视频人物场景 |
| 声场参照 | 提供声音/配乐参照 | 配合整体氛围 |

### 固定角色 / 固定场景建档（来自剧本医生的资产清单，全程复用同一段）
- **固定角色**：年龄+身份+发型+妆容+服装+眼神气质，结尾加「全程固定五官、发型、穿搭不变」。
- **固定场景**：空间+背景陈设+光线+**环境音**（"烛火爆裂声、更夫锣声""空调低鸣、键盘敲击"）。

## 二、空间走位块（AI 会幻觉空间，必须绝对量声明）

- **何时声明**：2+ 角色同框、关键道具在特定表面、相机几何重要时。
- **写法**：
```
⚠️空间布局（MAIN VIEW=从[方向]俯视）：
位置A：@image4 站中央通道最前、面朝三人。
位置B：@image1 与 @image2 通道中并肩，距 @image4 约 3 米，面朝其方向。
位置C：@image3 在 B 正后方 1.5 米——严格在其背后、被部分遮挡。
⚠️[距离、轴线、遮挡、身高/视线等模型易错处]
```
- 必写：**米数 + 朝向（东西南北/相对主视轴）+ 谁遮谁 + 视线锁定**。
- **参考图方位锚定法**（实拍有效）：直接说「走廊在参考图右上角」「相机放冰箱旁、画面右侧瞥见冰箱、深入房间」，比抽象描述更易命中。位置改了就重画俯视图再改提示词。

## 三、延展 EXPAND / 组接 COMBINE

| 延展（长镜·沉浸） | 组接（短镜·冲击） |
|---|---|
| 对话推进、情绪层层递进 | 高速剪辑、视角频繁切换 |
| 单一空间连续位移 | 跨场景大幅时空跳跃 |
| 适合 30-45 秒不间断叙事 | 拆成 10-13 秒独立片段 |

- **核心原则**：**文戏优先「延长」，武戏优先「拼接」**。
- **延长写法**：`将 @视频1 向后平滑延长 N 秒，保持角色外观、镜头方向、动作节奏和空间关系一致。`（必要时加 no watermarks, no subtitles）

### 几条镜头合成一条 15 秒
- **同角色集 + 同地点 + 连续情绪/时间 + ≤15 秒**才合并。
- 遇下列就拆开：跨地点硬切、角色进出改 handle、画幅/焦段切换、需独立 15 秒包络的情绪长镜、插入道具/屏幕特写。
- 拿不准**宁可多条短包络**——Seedance 对紧凑提示词比超载提示词处理更好。

---

## 四、局部与首尾帧编辑

- 在精确时间坐标 + 空间区域下做「精准手术」：替换人物/调道具/修画面，保持其余不变。
- 善用**首尾帧锚定**配合「帮我改这个人」等局部指令。

## 五、运动优先写法（轻运动 · 环境先动 · 保持一致）

多图参考下，静态长相已由参考图锁定，提示词的增量价值在"动"。借鉴静帧方法论第三原则——**优先轻运动、环境先动、镜头缓慢、动作连续不跳变**：

- **先让环境动**：雨丝下落、烟尘漂浮、火光闪烁、金属边缘高光流动、湿地反光、空气微粒——最容易出质感且不易崩。
- **再让主体小幅动**：缓慢抬手、轻微转头、衣摆/发丝被风动、指节收紧、身体微倾、眼神缓移。
- **镜头缓慢动**：缓推/轻微上移/低角度缓近/保持稳定/轻微手持，避免快速旋转。
- **保持一致 + 连续**：写"保持资产的身份/服装/道具/构图比例一致""动作连续、不突然跳变"。复杂大动作要么拆更多镜头、要么用首尾帧锚定，不在一条里硬塞。
- ⚠️这是**多图参考**下的运动写法，不是"在某张原图上往后续帧动"的首帧逻辑——别写成"让这张图动起来"，而是"这组参考下，画面如何小幅、连续地运动"。
