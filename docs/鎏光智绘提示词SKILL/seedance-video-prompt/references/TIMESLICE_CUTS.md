# 时序切片 · 切点规则 · 引擎硬约束（多镜合并的核心规则）

> 提炼自 Seedance 2.0 实战资料（seedance-director 引擎规则 / 核心使用方案 / 提示词写法长图文）。
> 配合 SKELETONS.md 的统一骨架使用。

## 一、时序拆解（THE FOUNDATION）

把片段叙事切成时间片，**每片锚定四要素：主体 / 空间 / 动作 / 视角（景别+运镜）**。

- ❌ 笼统写法："一位咖啡师在清晨的咖啡馆冲手冲，猫趴在柜台上。"（无镜头感）
- ✅ 切片写法：0-5s 中景固定·拿起手冲壶 → 5-10s 特写缓推·螺旋注水 → 10-15s 近景横移·擦拭壶身（猫入画）。
- 远景固定镜头收尾（10-15s 段）非常有掌控感，适合作片段终拍。

## 二、片段划分（哪些镜合成一条提示词）

**合并条件（全部满足才合并）**：同一角色组在画面中 + 同一场景/场景子区 + 情绪/时间连续（无跳时、无大情绪转轴）+ 总时长 ≤15 秒 + 中文 ≤3000 字符。

**拆分时机（任一命中就另起一条）**：跨场景硬切；主要角色进/出场改变 handle 列表；需要独立 15 秒情绪弧的反应戏（七拍情绪弧值得独享一条，别和动作戏挤）；道具/屏幕特写插镜（独立 ECU 一条）。

**节奏基准**：没有固定比例，按戏决定——悲恸崩溃的 3 拍可收成 1 条（不要把哀恸剪碎）；7 个不同机位/焦段/情绪拍各自成条。拿不准时**宁多条短封装，勿一条塞爆**。

## 三、切点规则

1. **双对比（强制）**：每次切换必须同时改变**景别**（极远→远→中→中近→近→特写→大特写）和**机位性格**（手持 / 固定 / 稳定跟拍 / 摇臂升降 / 航拍——不连续重复同一种）。
2. **重新锚定 + 180° 轴线**：切回已建立的空间后，重述谁在哪、面朝哪；切前向左移动的人切后仍向左。运动方向显式写出。
3. **插镜（insert）**：0.3-0.5 秒的戏剧标点，任何景别；**不承载剧情节拍**（只能是静态瞬间）；**必须有因果动机**（主角被砸上引擎盖→切**他的**手抠住金属；不要无来由的靴子踩水）；**写明是谁的**身体部位/物件，不署名 Seedance 会画错。
4. 不写逐镜秒表化的节拍控制到一镜到底的段落里——想切就用时间码，不想切就写「单镜头(one-shot，无剪辑，不要分段或定义节拍)」。

## 四、引擎硬约束（违反=画面崩）

- **动作写意图+招式名，不写生物力学**：✅"旋身后踢命中"；❌"左前臂旋转45°在腕部格挡右勾拳"。用户点名的招式保留；遇到关节级机械描述压缩成招式名或意图。
- **写力与方向，不写破坏序列**：✅"被撞上车身，金属凹陷"；❌"摔向车门、玻璃碎裂、借反弹扫腿"。
- **跨切稳定追踪 ≤3 个角色**：每镜写明"谁对谁、互动向量"。
- **出框 = 隐性剪辑**：角色离开画面即本镜内消失，不在同一连续镜头里出框又回来。
- **画外 = 不存在**：状态变化必须先在镜头内呈现，才能被后文引用。
- **禁反射镜头**（刀面/水洼/镜子叙事）——动起来破坏几何。
- **只描述能看见/听见的**：❌"空气里有松木味"；✅"满地松针，风穿过枝叶"。
- **微表情写成物理**：✅"咬肌收紧、鼻翼扩张"；❌"看起来很愤怒"。

## 五、动作节奏（SLOW & CONTINUOUS）

- **写小幅动作**：小幅动作更易精准还原（缓步前行、轻抬手臂、微颔下颌、自然落座）。
- **承接余势**：动作从静止顺滑滑入下一拍，连续性自然流淌。
- 修饰词用「缓慢、猛然、持续、连续」，避免「高速、剧烈」这类模糊词；动词要具体（撕裂/刺破/炸裂/闭合/震动/弹射）；可加声音细节（骨骼脆响、布料摩擦）。
- **复杂编排让模型自己来**：给方向词（如"剑舞收势"）优于逐步微操。

## 六、多参考配比（外部素材时）

形象参照 1-2 张 + 环境参照 1 张 +（可选）动态参照 1 段 +（可选）声场参照 1 段，共 4-5 个为稳；素材不是越多越好。动态参照学的是**运动轨迹**，不是复刻原片人物场景。

## 七、延展与组接（片段之间）

- **文戏优先「延长」**：`将 @视频1 向后平滑延长 N 秒，保持角色外观、镜头方向、动作节奏和空间关系一致。`
- **武戏优先「拼接」**：拆成 10-13 秒独立片段分条生成，剪辑时组接出冲击。
- 局部修改用首尾帧锚定做"精准手术"，其余画面保持不变。

## 八、避坑

复杂多人互动/打架跑跳剧烈/大幅舞蹈难稳定；「好看/很美/很酷」无效；「超高速+极度稳定」自相矛盾；色情暴力危险动作版权人物违规。

## 九、in medias res

默认戏已在进行中开拍（除非明确"从…开始/结束于…"）。开场不要从静止摆姿势起。
