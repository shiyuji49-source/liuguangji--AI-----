# 运镜与表演

## 一、运镜公式库（6 种常用，填空即用）

| 运镜 | 模板 | 适用 |
|---|---|---|
| 后方跟随 Dolly Follow | 后方跟拍，镜头紧随[人物]穿过[场景]，低角度平稳推进，背景飞速延展 | 追尾进场/奔跑/探索，强代入 |
| 环绕旋转 360° Orbit | 360° 环绕，以[主体]为中心缓慢旋转，展现[场景]全景，光影交替，主体居中 | 高光时刻/孤独感 |
| 缓慢推进 Slow Push-In | 镜头缓慢推近，从[主体]中景渐移至局部特写，背景渐虚 | 放大情绪/关键线索 |
| 低角度仰拍平移 Low Angle Pan | 低角度贴地平滑移动，向上仰视[人物/建筑] | 主体高大/史诗感 |
| 上帝视角俯瞰 Bird's Eye | 垂直俯瞰，从高空下拍[场景]，缓慢旋转拉高，几何构图 | 全局观/交代关系 |
| 侧位横移 Lateral Tracking | 侧面平行跟拍，与[主体]同速移动，前景遮挡制造视差 | 街道行走/车内，空间深度 |

**一镜一运动**：一段里只用一个运镜；先定景别，再定唯一运镜，多个混用会乱。

## 二、运镜 = 焦点角色情绪的「替身」

| 情绪 | 运镜写法 |
|---|---|
| 愤怒/紧张 | 不安抖动手持——呼吸式漂移、上下微抖、左右轻摆，不规则，禁稳定器 |
| 平静/掌控 | 平稳流畅手持——只剩极细微规律呼吸式微动，禁稳定器 |
| 悲伤/脆弱 | 缓慢低位手持——呼吸放慢，画面极轻微下沉 |
| 震惊/揭示 | 先严格静止，0.5 秒后极缓推进/拉远，总位移 ≤15cm |
| 动作 | 60fps、180° 快门运动模糊，禁超出快门范围的拖影 |
| 终镜/定论 | 正上方俯拍 top-shot，0.3-0.5 秒 freeze frame，时间静止 |

> 情绪在一镜内变化时，相机分阶段同步变化，每个相机阶段绑到表演节拍 ①②③。

## 三、焦段选择

| 用途 | 焦段 | 光圈 |
|---|---|---|
| 极致情绪大特写（额头到下巴填满） | 85/100mm | F1.4 |
| 中景对白/双人 | 50mm | F2.0-2.8 |
| 广角/确立 | 35mm | F4-5.6 |
| 道具插入特写（锁焦） | 50/85mm | F1.4 |
| 微距（毛孔/水珠/织物） | 45mm macro | F2.8 |

## 四、剪辑规则

- **双重对比（强制）**：每个 cut 同时改变**景别**与**相机模式**。
  - 景别梯度：极远→远→中→中近→近→大特写。
  - 相机模式：手持/静止锁定/稳定跟拍/摇臂升降/航拍——跨 cut 不重复。
- **重锚定与 180° 轴线**：剪回旧空间后重述谁在哪、朝哪；cut 前左→右移动，cut 后保持同向。
- **插入镜**：0.3-0.5 秒、无剧情、需因果动机、**指明是谁的身体部位/细节**（不标主体会渲染错），并遵守双重对比。
- 节奏由描述密度体现，输出里不写每镜时间码（B 级快剪用时间码除外）。

## 五、景别速记

WS 远景 / MS 中景 / CU 近景 / ECU 大特写 / MACRO 微距 / PAN 横纵摇 / OS 画外音 / VO 旁白 / DISSOLVE 叠化。

## 六、表演：情绪词打底 + 物理细节点睛

少写抽象情绪标签，多写可观测身体细节。**写法**：情绪词打底（"心碎""强忍"）+ 关键处补 1-2 个物理细节（下唇颤抖、眼眶泛光但泪不落）。

### 微表演物理目录（按情绪拆）
- **愤怒/决意**：咬肌搏动、颈动脉脉搏、太阳穴血管浮起、重音词鼻翼扩张、瞳孔收紧、高潮不眨眼。
- **焦虑/紧张**：喉结一次吞咽、台词前短促鼻吸、舔润下唇、脸颊泛红、关键词瞳孔扩张（除非明说否则不哭）。
- **悲伤/脆弱**：眼角外缘下垂、眼睛湿润带眼神光但不淌不闪、眉头中央皱起、嘴唇轻颤、头微低。
- **控制/优越**：呼吸平稳、手指放松、缓慢眨眼、下颌微抬、Duchenne 微笑（逐渐建立，非一开始就完成）。
- **沉重**：双肩下沉、头微垂、深缓呼吸、声音更低闷、回答时头侧倾 5-15°。
- **震惊/冻结**：开镜冻结 0.3-0.5 秒、瞳孔扩张、嘴唇微张无声、freeze 后延迟急吸、视线锁定触发源不眨眼。
- **压抑（最难）**：每块面部肌肉对抗涌上的情绪、咬肌慢收、喉结用力吞咽、眼眶积泪泛光但不落、缓慢控制的深吸（胸腔起伏）、下颌微抖一次立刻收紧。
- **惊讶 4 变体**：轻微正面（眉扬+唇微张+瞳孔 0.3s 扩张）/ 震惊（急吸+睁大+冻结+手抬胸前）/ 难以置信（缓慢眨眼+头侧倾+唇收紧+单眉抬）/ 惊喜（眼睛点亮+笑容逐步建立+双肩放松）。

### 台词与多角色
- **一句台词 = 一个镜头**：每个角色台词只出现在其特写内；每句写明**对谁说**。
- **台词三拍**：台词前（吞咽/吸气/舔唇）→ 台词中（哪些词用鼻翼/语调/瞳孔强调）→ 台词后（约 0.5 秒屏息再动）。
- 台词写引号内、模型做带情绪节奏的口型表演；**保留原文不翻译**。
- **旁观者在虚焦里也不静止**：写其头/视线朝向与微动；多人反应**错峰 0.3-0.5 秒**，绝不完美同步。

### 表演禁忌
❌「站着干说话」（永远有微动）；❌ 夸张鬼脸；❌ 思考时望天花板（指明视线方向）；❌ 默认流泪（除非剧本明确）；❌ 笼统「看起来很 X」。
