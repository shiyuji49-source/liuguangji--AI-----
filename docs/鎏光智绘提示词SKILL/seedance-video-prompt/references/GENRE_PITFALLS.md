## 二、翻车点 → 解法

| 翻车 | 解法 |
|---|---|
| 相机位置/朝向错、凭空多门、分不清走廊 | 加空间布局块 + 参考图方位锚定（见 ASSET_SPACE_EDIT）|
| 写每 3 秒节拍 → 被迫乱切 | 明确「一个连续镜头，不要分段/定义节拍」|
| 掉帧（要 24fps 出 ~12-18fps 顿挫）| 提示词强制 24fps；画面够好则在剪辑里逐帧删重复帧（音轨同被切，只当 B-roll）|
| 拿错道具/脸崩/换错手 | 目标道具单独作参考图标明「这是 image4」；明确「右手」「物体留在原处」；提示词过长就**整条重写**|
| 报 NSFW 误判 | 改提示词措辞重试（图过了 IP 检查则问题在提示词）|
| 同图反复编辑丢细节 | 改用擅长局部编辑的图像模型，或保留布局从头重生成 |

**实战要诀**：先把**场景图做到完美**（视频灯光/风格多继承自它，场景糊视频必糊）；**批量+跳看**（一次批 8 个，构图对了才看完整段；几个犯同样错就停下回头改提示词；失败生成不扣 credits）；**介绍角色两次**（一次"她是谁"静态、一次"她能干什么"动作）；**复杂动作给一个真方向词**（如"K-pop 编舞"）比逐步微操好；**边剪边修**（占位音乐、L-cut）。心理预期：约每 64-100 个生成出 1 个能用。

## 三、避坑红线

- **复杂动作**：多人互动、打架、跑跳剧烈、大幅舞蹈——难稳定，必要时拆更多镜头/多用首尾帧锚定。
- **模糊词**：「好看/很美/很酷」无法有效引导，删。
- **矛盾要求**：超高速 + 极度稳定（难同时满足），别绑在同一瞬间。
- **违规内容**：色情、暴力血腥（转非直白：战损/暗红污渍/侧面构图/光影暗示）、危险动作、版权人物（不拿明星 1:1 还原）——这些理应在**剧本医生阶段**已处理，此处再兜一次底。

## 附录 · 中英镜头语言对照（主输出中文，术语备查）

- **角度**：仰拍 low-angle / 俯拍 high-angle / 荷兰角 dutch / 鸟瞰 bird's-eye / 蚁视角 worm's-eye / 平视 eye-level / 过肩 OTS
- **焦段**：广角 14-24mm / 标准 35-50mm / 长焦 85-200mm / 微距 macro
- **运动**：跟拍 tracking / 推 dolly-in / 拉 dolly-out / 摇臂 crane / 横摇 pan / 纵摇 tilt / 甩镜 whip-pan / 环绕 orbit / 推进 push-in / 后拉 pull-back / 手持 handheld / 斯坦尼康 Steadicam / 航拍 aerial
- **时间**：升格 slow-motion / 变速 speed ramp / 定格 freeze frame
- **转场**：硬切 hard cut / 匹配剪辑 match cut / 甩镜转场 whip-pan / L 型剪辑 L-cut
