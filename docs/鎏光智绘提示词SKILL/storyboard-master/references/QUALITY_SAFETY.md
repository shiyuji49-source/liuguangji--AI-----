# 质量控制 · 安全表达 · 统一避免清单

## 统一避免清单（全 skill 唯一一份，§5/§11 都引用它）

海报站姿；完整资产展示；均匀轮廓光；全画面平均高清；人物无动作摆拍；背景与主体无关系；过度锐化；廉价滤镜感；CG 塑料感；满屏粒子与无来源特效光；脏黑、灰膜、过量噪点；假脸（磨皮+无来源补光）；杂乱背景；站位不清；缺前中远景；互相冲突的光线与色彩；古装写成网游仙侠（蓝金发光、满屏飞叶、过锐武器、悬浮特效、刺绣陈列、山水壁纸、高清武器展示）。

## 输出前静默质检（复用 §4 九条判断，重点核对）

是否有明确视觉中心；是否有可见事件或人物关系；机位是否清楚；主光来源是否清楚；颜色是否落在可信载体；核心区域是否最可读、次要区域是否主动退后；道具/场景/造型是否真正改变画面；负面控制是否少而准确；是否避开上面的"统一避免清单"。

## 常见问题 → 修复方向（先修结构，再考虑加风格词）

- **画面海报化** → 改为未完成动作、偏斜观看、遮挡、现场压力或偶然裁切。
- **CG 资产感** → 减少完整结构展示、均匀轮廓光、材质陈列与灯效堆砌。
- **脏黑或太锐** → 删灰膜、噪点、过量烟尘、硬边、微对比。
- **人脸假** → 取消磨皮与无来源补光，让脸服从现场光。
- **主体不突出** → 强化动作、观看关系、光线或清晰度层级。
- **人物贴背景** → 增加接触、遮挡、距离、运动方向或局部互动光。
- **摄影像广告** → 恢复观看距离、现场色偏、地点表面与偶然裁切。
- **古装像网游** → 减少蓝金光、满屏飞叶、过量刺绣、过锐武器与悬浮特效。

**只修当前问题，不重写用户已满意的部分。**

## 安全表达与平台适配

剧情涉及血腥、尸体、伤口、酷刑、极端暴力时，转为影视可用的非直白表达：战损痕迹、暗红污痕、破损衣甲、尘土与雨水混合痕迹、远景模糊的倒伏身影、压抑的战后气氛、破碎兵器与凌乱旗帜、不直视伤口的侧面构图、以光影和环境暗示创伤。避免露骨、猎奇、过度血腥的提示词。（与道具/服装 skill 的"血迹→暗红污渍"口径一致。）
