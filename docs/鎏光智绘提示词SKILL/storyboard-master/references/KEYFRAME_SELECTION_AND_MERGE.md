# 关键帧筛选与合并

## 同类镜头合并

同一场景中，多个镜头若属于类似景别、镜头方向、人物站位、光线关系，**只产出一段关键帧提示词**，不要重复生成大量几乎相同的提示词。

两个镜头若满足以下**三项以上**，合并为一段：同一场景 / 同一人物或人物组 / 同一景别 / 同一镜头角度 / 同一机位方向 / 同一光线方向 / 同一空间层次 / 同一情绪状态 / 同一道具使用方式 / 同一叙事功能。

合并时选**最有画面张力、最适合作为视频首帧或参考帧**的一刻。例：

- 同一人物在同一位置连续说话、都是中近景平视正面 → 一段。
- 同一场景多个相似反应镜头、构图方向一致 → 一段。
- 同一组人群同一机位下的议论、惊讶、后退 → 合并为一段群像关键帧。
- 同一角色连续动作、构图与光影变化不大 → 只选最有戏剧张力的一帧。

## 关键帧筛选

优先生成关键帧的画面：场景建立镜头；主要人物首次入场；人物关系变化；情绪转折点；剧情信息揭示点；重要动作开始或完成的瞬间；有视觉隐喻 / 强构图价值的画面；重要道具出现或被使用；光影、空间、气氛明显变化；可作视频首帧/中段参考帧/结尾定格帧；能交代轴线、人物方向、视线关系；能给后续视频提供稳定光影基准的画面。

**不要把所有台词都变成画面。** 只有当台词引发人物状态、关系或剧情信息变化时，才生成关键帧。

## 单场多关键帧

单场可含多段关键帧，但每段必须有明确差异（满足其一即可）：景别 / 机位方向 / 人物关系 / 空间层次 / 光影重点 / 情绪状态 / 叙事功能 / 关键道具 / 画面压力方向 / 视频参考用途不同。**只是台词不同、画面差异不大的，不要重复输出。**

## 关键帧类型（按需选，不必每场全用）

- **A 场景建立帧**：空间、时代、光线、空气、场面规模、环境气氛。
- **B 人物关系帧**：站位、视线方向、权力关系、情绪距离、对峙或亲疏。
- **C 动作节点帧**：关键动作瞬间、肢体姿态、移动方向、兵器/道具使用方式。
- **D 情绪爆点帧**：表演、眼神、表情、压抑/崩溃/愤怒/震惊/隐忍/决绝。
- **E 道具 / 细节帧**：关键道具、手部动作、服装细节、场景暗示信息。
- **F 转场 / 隐喻帧**：视觉象征、匹配剪辑、光影变化、空间压迫关系。
- **G 群像调度帧**：群演数量感、站位层次、视线方向、集体情绪，以及主体被围绕/凝视/避让/孤立/压迫的关系。
