# 静帧提示词方法论（注意力分配 · 构图原型 · 冲突自检 · 静帧取舍）

> 来源：用户《AI 静帧提示词写法·7 条视频总结》。本文件是给静帧（image2）提示词的**写法增强**，与 24 字段配合：24 字段管"写什么"，本文件管"**怎么分配权重、用哪种构图、别自相矛盾、哪些镜头才值得出静帧**"。

## 一、核心：提示词 = 分配模型注意力（权重 > 顺序）

AI 不按从前到后逐条执行，而是在多个信号间做**权重竞争**。写提示词只做三件事：

- **定主权重**：这帧最不能丢的是什么（主体 + 核心动作/关系），写得最具体。
- **降干扰权重**：辅助信息（背景、氛围）别抢主体，退为低频形状/焦外/暗部。
- **避冲突权重**：互相打架的描述删掉或改写（见第四节）。

具体词的权重 > 抽象词："双手合十" > "虔诚"；"抬头望向佛像" > "心生敬畏"。这条是 §4「分配清晰度层级 / 删无改变元素」的上位原则。

## 二、抽象词必须拆成可见元素

抽象词可留，但**必须配套具体可视动作**，否则模型乱猜：

- **情绪词** → 眼睛/眉毛/嘴角/呼吸/停顿/手部（详见 `PERFORMANCE_SPACE_COMPOSITION.md` 表演转化）。
- **氛围词** → 光线/烟雾/天气/材质/背景层次。
- **风格词**（"电影感"）→ 镜头/构图/色调/景深/运动方式（别只写 cinematic）。

## 三、构图原型词汇（补 24 字段【景别/构图】的缺口）

24 字段有景别/机位/前中远景，这里补"画面重心"用的构图法，**每帧选一个主导构图**：

| 构图 | 作用 | 适合 |
|---|---|---|
| 居中 centered | 仪式感、稳定、直接 | 神像/佛像前人物、机甲正面、对峙 |
| 近似对称 near-symmetrical | 左右视觉重量接近，稳而不死板 | 宫殿、寺庙、正面人物、仪式 |
| 三分法 rule of thirds | 自然、电影化、叙事 | 望向远方、旅途、孤独、横向场景 |
| 引导线 leading lines | 用道路/光/刀剑/建筑线引导视线 | 走廊、桥、山路、光束、武器 |
| 低角度 low-angle | 主体高大、压迫、力量 | 机甲、武士、英雄、巨像 |
| 特写 close-up | 突出表情/眼/手/细节 | 微表情、情绪变化、握道具 |
| 远景 wide shot | 环境与人物关系 | 史诗、寺庙、废土、城市、背影 |

**构图公式**：主体位置 + 画面重心 + 视线方向 + 前中后景关系 + 镜头距离。

## 四、内部冲突自检（出每帧前过一遍）⚠️

互斥要求会让模型折中、漂移或崩坏。常见冲突 → 解法：

| 冲突 | 解法 |
|---|---|
| 背对镜头 + 看清正脸 | 改三分之二侧背/侧脸/轻微回头，只露紧绷嘴角与低垂眼神 |
| 完全静止 + 复杂动作 | 主体稳定，只手部/头部轻微动 |
| 远景 + 丰富微表情 | 要表情就改中近景/特写 |
| 黑暗 + 看清全部细节 | 低调光，但主体脸/轮廓有局部补光 |
| 写实 + 二次元 | 只留一个主风格 |
| 快速运镜 + 细节稳定 | 改缓慢推进/轻微手持/稳定镜头 |
| 多主体 + 每个都清楚 | 定一个主角，其余退为陪衬/背景 |

**自检问**：这帧主体是谁？第一眼看哪？动作能被看见吗？情绪有具体表情支撑吗？构图与镜头距离匹配吗？参考图与提示词冲突吗？有没有要求 AI 同时做两件互斥的事？背景会抢主体吗？

## 五、静帧取舍（哪些镜头才值得出静帧）⚠️本次新增

静帧是**强控手段**，不是每个关键帧都要出。按**项目分级 + 镜头价值**取舍：

- **S / A 级（精品）→ 做静帧，但选择性**。优先给下列镜头出静帧：
  - 复杂构图 / 多人同框 / 难控动作（打斗、关键姿态）；
  - 关键情绪爆点、关键道具特写；
  - 后续会被**反复参考**或要严格锁住的镜头（主角定调帧、场景建立帧）。
  - 其余镜头只做镜头规划（24 字段或简述），**不必生成静帧**。
- **B 级（跑量剧）→ 静帧可选、不固定**，由用户/项目决定：
  - 不做静帧时，该镜头**跳过 image2 静帧、直接进 Seedance**（资产参考 + 八要素骨架）；
  - 要做也行，按上面取舍只挑极少数最关键镜头。
- **取舍原则**：一帧静帧若能省下后面几十次视频试错就值得做；只为"好看"不值得。

> 一句话：静帧管的是"**哪几个镜头需要强控**"，不是"把每个镜头都画出来"。下游用静帧时也是**选择性参考**（不是每条视频都喂静帧）。
