# 输出格式 · 24 字段 · 可直接出图提示词

## 两种模式

- **默认：分镜静帧导演模式** —— 每段关键帧 = **24 字段导演分解** + 末尾 **【可直接出图提示词】**。默认任务不要擅自取消 24 字段。
- **紧凑机器提示词模式** —— 仅当用户明确要"不要字段 / 只要连续提示词 / 机器提示词版 / 紧凑版"时，输出 2–4 个自然段的连续提示词（结构同下方"可直接出图提示词"的四段）。

## 输出骨架

```
# 第 X 集 · 分镜静帧关键帧提示词

## 场景 X-X · 场景名称 · 时间 / 内外
【场景功能】一句话说明该场景在剧情中的功能。
【资产绑定】人物资产 / 场景资产 / 道具资产 / 服装资产 / 文本推导资产（没有资产时列推导元素）。
【本场关键帧数量】本场共 X 段，已合并类似景别、镜头方向、人物站位的重复镜头。

### 关键帧 1 · 名称
【关键帧功能】场景建立 / 人物关系 / 动作节点 / 情绪爆点 / 道具细节 / 群像调度 / 转场隐喻。
【对应剧情】一句话概括该帧的剧情节点。
【使用资产】点名本帧要带进 image2 的参考图（如：人物=杨延昭定妆照；场景=金殿日景图；道具=断枪图）。
【静帧提示词】（下列 24 字段）
…… 24 字段 ……
【可直接出图提示词】（把 24 字段压成一段 image2 自然语言成品）
```

## 24 字段（每段关键帧逐项填写）

1. **镜头角度**：平视 / 低角度 / 高角度 / 俯拍 / 仰拍 / 侧后方 / 贴近观察。
2. **镜头尺寸**：特写 / 近景 / 中近景 / 中景 / 中远景 / 远景。
3. **景别**：画面收纳范围，让主体与环境关系清楚，**并写明画幅比例**。
4. **景深**：哪里清晰、哪里虚化（避免全画面平均清晰或平均模糊）。
5. **电影机 / 胶片质感**：电影截图感、胶片颗粒、镜头柔和度、真实摄影质感。
6. **主体**：画面中心人物 / 物体 / 场景核心。
7. **核心身份 / 物种**：主体身份、年龄气质、角色类型。
8. **体型 / 外貌特征**：体型、轮廓、脸部气质、发型、姿态辨识度；**有资产时强调"保持资产人物特征"**。
9. **材质 / 纹理**：皮肤、布料、金属、玻璃、雨水、木头、石砖、纸张、茶水、兵器等真实质感。
10. **动作 / 姿态**：主体正在做什么；身体紧张 / 放松 / 犹豫 / 压迫 / 退缩 / 爆发 / 克制。
11. **核心机位位置**：摄影机站哪看（殿门口 / 长街尽头 / 屏风侧面 / 桌案低位 / 人物肩后 / 巷口斜侧）。
12. **天气与氛围**：晴 / 雨后 / 夜 / 雾 / 潮湿 / 冷清 / 温暖 / 压迫 / 肃杀 / 烟火气。
13. **前景**：靠近镜头的虚化物 / 遮挡物 / 人物局部 / 道具边缘 / 门框 / 帘幕 / 树影 / 烟尘 / 光斑。
14. **中景**：主体与主要动作所在区域。
15. **远景**：远处空间 / 街道 / 城门 / 殿柱 / 屏风 / 天空 / 墙面 / 群演 / 环境延伸。
16. **背景**：背景如何服务主体，避免无意义杂乱。
17. **主光源 / 方向**：窗光 / 殿门光 / 天光 / 火光 / 烛光 / 顶光 / 屏幕光 / 晨光 / 夕阳。
18. **光线软硬质感**：柔和 / 硬朗 / 冷淡 / 扩散 / 潮湿反射 / 边缘清晰。
19. **辅光源 / 互动光**：环境反射、地面/墙面反光、火盆反光、衣料反光、道具反光、脸部补光。
20. **特效光 / 气氛光**：体积光、雾光、烟尘、雨水反光、空气尘埃、烛火边缘光。
21. **整体色温 / 基调**：冷 / 暖 / 中性 / 蓝灰 / 金黄 / 青绿 / 低饱和 / 肃穆。
22. **色彩饱和体系**：主色与辅助色，避免颜色互相打架。
23. **高光与阴影倾向**：高光是否克制，阴影是否柔和，黑位是否干净。
24. **对比度与饱和度**：整体对比与饱和，并加清洁要求（避免脏、糊、过锐、塑料感、杂乱噪点、混浊色彩）。

## 【可直接出图提示词】写法（image2 自然语言成品）

把上面 24 字段压成连贯的四句/四段，可直接粘进 image2，**配合本帧【使用资产】列出的参考图一起用**：

1. 主体 + 核心身份 + 动作/姿态 + 事件或关系 + 机位位置 + 景别与镜头尺寸 + **画幅**。
2. 体型/脸部气质特征（有资产则写"保持资产人物特征"）+ 关键材质 + 前/中/远景空间层级 + 主动退后的区域。
3. 主光方向 + 光软硬 + 辅光/互动光 + 特效气氛光 + 色温基调 + 颜色载体 + 清晰度层级。
4. 少量最相关的失败防护（避免……）。

写法用自然语言中文；画幅写成"16:9 横屏影视剧静帧构图"之类；规避项写"避免……"，不用 `--ar`/`--no`。

## 命名规则

关键帧名称要短、准、有导演感：建立（皇城大街清晨建立帧）、入场（杨延昭远景入场帧）、关系（君臣隔屏对峙帧）、情绪（幼童仰望惊惧帧）、动作（断枪触地特写帧）、道具（茶盏落水审判帧）、隐喻（金殿阴影压顶帧）、群像（百姓惊散群像帧）。不要用"画面一""镜头二""好看的场景"这类空泛名。

## 范例（仅示意格式，实际按导入剧本生成）

### 关键帧 · 君臣隔屏对峙帧
【关键帧功能】人物关系帧。
【对应剧情】杨延昭在金殿被屏风后的国师隔空质问，权力压迫成立。
【使用资产】人物=杨延昭定妆照、（屏风后）国师定妆照；场景=金殿日景图；道具=断枪图、屏风。
【静帧提示词】
镜头角度：低角度仰看｜镜头尺寸：中景｜景别：纳入屏风与殿柱、交代隔绝关系，16:9｜景深：人物清晰、屏风后人影与殿柱退入焦外｜质感：电影截图感、细腻胶片颗粒｜主体：殿心而立的青年将领｜核心身份：边疆归来的杨家将后人，肃然、克制｜体型外貌：保持杨延昭定妆照的脸型、发型、年龄感与气质，劲瘦挺拔｜材质：旧甲皮革、暗铁断枪、屏风木格与绢面｜动作姿态：握断枪、肩背端正、指节微紧的隐忍｜核心机位：屏风侧前方、略低于人物｜天气氛围：殿内肃杀、空气微尘｜前景：虚化的屏风木格一角｜中景：持枪而立的杨延昭｜远景：屏风后模糊人影与高大殿柱｜背景：暗木殿堂退为低频暗部｜主光：高侧殿门光自画面左上斜入｜光软硬：偏硬、边缘清晰｜辅光：地面青石的微弱反光补脸暗部｜特效光：殿门光柱里的体积浮尘｜色温基调：冷调青石灰为主｜色彩体系：青石灰+暗木为主，断枪暗红绳与屏风一点金为锚点｜高光阴影：高光克制、阴影柔而黑位干净｜对比饱和：中对比、低饱和，画面干净不脏不过锐。
【可直接出图提示词】
16:9 横屏影视剧静帧，低角度仰看的中景：一名边疆归来的青年将领独自立于金殿殿心，手握一柄断枪，肩背端正、指节微紧、神情隐忍克制，屏风侧前方略低的机位，交代他与屏风后之人的隔绝对峙关系。保持人物资产中的脸型、发型、年龄感与气质，劲瘦挺拔；旧甲皮革与暗铁断枪、屏风木格与绢面质感真实；前景是虚化的屏风木格一角，中景是持枪的将领，远景是屏风后模糊人影与高大殿柱，暗木殿堂退为低频暗部。主光为高侧殿门光自左上斜入、偏硬且边缘清晰，地面青石微弱反光为脸部暗部补光，殿门光柱里有体积浮尘；冷调青石灰为主，断枪暗红绳与屏风一点金作为颜色锚点，核心人物最清晰、屏风后与殿柱退入焦外。避免：海报站姿、均匀轮廓光、全画面平均高清、CG 塑料感、过锐、假脸、杂乱背景、网游仙侠的蓝金发光。（出图时把杨延昭定妆照、金殿日景图、断枪图作为参考图一并喂入 image2。）
