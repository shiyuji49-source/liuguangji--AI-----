# 表演 · 空间 · 光影 · 构图

## 表演转化

剧本里的情绪必须转化为可视化表演，不要只写"愤怒""恐惧""隐忍""威严"。转成身体、眼神、呼吸、手部、肩背、重心、面部肌肉变化。例：

- **愤怒**：下颌绷紧，肩背前压，目光直刺对方，手指用力扣住衣袖边缘。
- **恐惧**：身体后缩，瞳孔放大，嘴唇微张，手指僵在半空，重心向后逃离。
- **隐忍**：眼神压低，唇线收紧，肩膀维持端正但指节微微发白。
- **上位者压迫**：人物端坐不动、几乎没有多余动作，镜头低位仰看，阴影压在下半张脸，周围人保持低头站位。
- **轻描淡写的权力**：人物动作平稳细小，手部继续完成茶、水、笔、印章等日常动作，而被审判者/弱势者处在画面边缘或阴影中。

## 空间与轴线

同一场景内保持空间轴线与光源方向一致。每段关键帧要明确：摄影机站在哪里、朝哪看、主体在画面哪一层、对手/关系对象在哪一侧、前中远景各承担什么功能、主光从哪来、反打时如何保持空间连续。剧情没有明确改变位置时，**不能随意改换空间方向**。

## 光影统一

同一场景内关键帧的光源方向必须一致，除非剧情明确发生时间、天气或空间变化。重点说明：主光源方向、光线软硬、辅光来源、互动光来源、体积光/烟尘/雾气/空气颗粒、高光控制、阴影层次、色温基调、颜色载体。**不要同一场景一会儿左主光、一会儿右主光**，除非是明确反打并说明光源相对方向。

## 构图机制（输入较短、画面不明确时主动建立，但不要写机制名称；只保留一个主导机制）

1. 门口/车门/窗边/屏风/殿门/走廊边界切开人物 → 等待、离开、关系断裂、权力隔绝。
2. 人物缩小在草坡/雪地/荒原/海岸/殿堂/长街/大面空间中 → 距离、仪式、孤独、时代压力。
3. 玻璃/镜面/水面/车窗/屏风/帘幕分割人物 → 沉默、监视、疏离、遮蔽真相。
4. 火光/爆闪/警示光/烛光/殿门光只照亮事件相关表面，其余退后。
5. 厨房/教室/小店/旅馆/茶室/街边/偏殿等普通地点保留生活证据，用低强度动作承载情绪。
6. 前景遮挡/门框切片/屏风透影/帘幕后影子/桌案低位等改变观看关系。

## 道具关键帧

道具有剧情作用时可单独成帧。必须说明：道具位置、材质、与人物手部关系、在画面中的构图权重、象征意义、光线如何落在其上、是否被前景遮挡或局部裁切、背景如何退后以突出它。不要只写"一个道具特写"。

## 群像关键帧

有百姓/士兵/宫人/食客/守卫/侍女/围观群众等群体时可成帧。必须说明：人群数量感、年龄层次、服装差异、站位层次、视线方向、情绪统一或分裂、主体是否被人群包围/避让/凝视/孤立/压迫、群演在前中远景分别承担什么功能。不要让群演变成杂乱背景。

## 参考图功能化转译

上传参考图时不要盲目复述所有细节，先判断每张图真正承担的功能（人物身份 / 脸型气质 / 发型 / 服装 / 姿势 / 场景结构 / 光线方向 / 颜色体系 / 材质质感 / 构图方式 / 氛围 / 镜头语言）。多图合并时只保留真正影响结果的继承项，并统一光线、空间与媒介。**不要把"参考图 1""保持原图风格"当主要描述，应转译为明确可执行的画面语言。**
