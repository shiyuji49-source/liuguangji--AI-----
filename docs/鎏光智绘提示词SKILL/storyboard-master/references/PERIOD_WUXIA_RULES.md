# 古装 / 武侠 / 宫廷 / 江湖 收束规则

处理中国古典、武侠、宋代、宫廷、边关、江湖、寺院、石窟、竹林、荒宅等题材时：

1. 画面以山水留白、衣袍线条、动作停顿、空间气韵和人物关系成立。
2. 不做仙侠网游感，不做古装写真感，不做蓝金过度发光。
3. 外景可用：山间天光、竹林漏光、芦苇逆光、湖岸薄雾、落日剪影、早春烟火气。
4. 室内可用：窗格光、门洞光、烛台、暗木反射、屏风透影、香烟体积光。
5. 主要环境色：竹青、墨绿、土褐、岩灰、米白、暗木、青石灰。
6. 小面积红色、金色、暗纹作为权力或行动锚点。
7. 山 / 竹林 / 芦苇 / 湖岸 / 街道 / 殿柱 / 屏风 只保留一条主导空间结构。
8. 人物通过停步、回望、持剑、围困、同行、错身、跪拜、端坐进入空间。
9. 剑 / 旗 / 茶盏 / 印章 / 屏风 / 烛台 / 门框 / 断枪 等，一次只让一个承担叙事中心。
10. 避免：统一青绿滤镜、复古黄滤镜、满屏飞叶、无重力轻功、刺绣陈列、山水壁纸、高清武器展示。
