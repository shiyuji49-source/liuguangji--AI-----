# Seedance 2.0 视频提示词要点（用于搭建"分镜 → 视频"提示词 skill）

> **来源**：Higgsfield 博客《Zephyr》案例拆解——一部全 AI 生成的科幻 K-pop 预告片，全程用 **Seedance 2.0**（字节系视频生成模型）制作，文章给出了每一步的提示词与用途。
> **本文做了什么**：把案例里可复用的"视频提示词写法"提炼出来，**去掉 K-pop 专属内容、剔除原文里不能照搬的露骨台词与血腥镜头**，并对接你现有的 image2 静帧套件。文中的示例提示词为按你古装题材**原创**，非博客原文。

---

## 一、它在你流程里的位置

你现有链路：剧本 →（人物 / 场景 / 服装 / 道具 / 群演 skill → image2 出静帧资产）→ 分镜大师（关键帧静帧提示词）→ image2 出静帧。

**Seedance 2.0 接在最后一环之后，把静帧变成视频（运动）**：

- image2 出的静帧 ≈ Seedance 的**首帧 / 参考画面**；
- Seedance 提示词在静帧基础上再补：**镜头运动、表演动作、物理与力、镜头切换、台词、音乐**。

所以这个新 skill 的本质 = 把"分镜大师的关键帧"再翻译一层，变成"Seedance 视频提示词"。

---

## 二、先锁资产，再动画（案例第一原则）

案例反复强调：**先做出一致的角色 / 世界资产，再生成任何一帧视频**。

- **角色三步法**：① 先定脸（同一段基础提示词反复生成，挑骨相 / 眼神 / 气质）；② 再单独定服装（任何重要细节——腰带、绷带、配饰——都在这一步写死，免得 40+ 个镜头里出问题）；③ 把脸 + 服装融合成一张**全身、白底、柔和电影光**的"角色母版图"。这张母版就是后面每次出场时模型比对的基准。
- **世界资产同理**：场景、怪物、机甲都先各自做成母版图，再开拍。一个有用的技巧：把两张你都满意的生成图融合，能得到一张你单靠提示词描述不出来的图。
- 案例用的工具是 Soul Cinema（脸 / 服装）、Nano Banana Pro（融合）、Cinema Studio（场景）。**对你而言这一步就是你已有的人物 / 服装 / 场景 skill + image2，技术要点可直接迁移，工具换成 image2 即可。**

---

## 三、一致性机制：母版图 + @ 标记资产

Seedance 保持"同一张脸 / 同一个场景 / 同一件道具跨镜头一致"靠两点：

1. 每个资产都有一张**母版图**（全身白底、柔和电影光）；
2. **每条视频提示词里用 @ 标记调用资产**——案例写成 `@角色名`、`@场景`、`@机甲`、`@怪物`，模型据此对齐外观。

对应到你的套件：@ 标记 = 你用 image2 做好的人物定妆照 / 场景图 / 服装卡 / 道具图。提示词里"保持资产特征"的文字仍是辅助，连续性主要靠**资产母版 + @ 调用**（和你分镜大师里"把参考图喂 image2"是同一思路，Seedance 只是用 @ 标签来点名）。

---

## 四、Seedance 提示词的核心写法（最关键的一节）

1. **写"摄影机怎么动"，不只写"画面里有什么"。** 案例大量描述机位行为：静止 → 突然一震、急甩镜、推进、绕拍、低角度大全景、撞击时的镜头抖动……把摄影机当成现场的人来写。
2. **把性格 / 情绪写进提示词。** Seedance 读得懂"傲慢、冷静自信、俏皮"这类性格语言，会给出对应的表演情绪。给对词，就给对情绪。
3. **写"物理与力"，不要写"字幕式动作"。** "机甲一拳打中怪物"是字幕、不是导演语言；要写力作用在身体上的**结果**——后坐力 → 头发被气浪掀起、手仍稳握操纵杆；重型机甲 → 沉重的踏地脚步、不轻飘。这样才有质量感、动量与"后果"。
4. **一条提示词里可以走多镜头。** 案例常在一次生成里覆盖"外部动作镜头 + 座舱内揭示"，靠明确切点串起来（"硬切到""视角随后转到""切回……"）。把这一组镜头的切换顺序写清楚。
5. **台词直接写进引号。** 角色可以说话，台词写在引号里，模型会做带情绪与节奏的口型表演。（涉及露骨 / 低俗台词时按"第七节·安全"处理，别照搬原文。）
6. **音乐 / 歌词作为"元素"先上传、再生成动作。** 跳舞 / 音乐段落：先把音乐和歌词作为元素喂进 Seedance，再生成画面——节奏、律动是"一起生成"出来的，不是后期配的。
7. **复杂动作"让模型自己来"。** 编舞这类，给一个真正的方向词（如"K-pop 编舞"）比逐步微操更好；过度指定每一步反而更差。

---

## 五、镜头规划原则

- **每个角色介绍两次**：一次确立"她是谁"（静态 / 座舱），一次确立"她能干什么"（动作）。案例上百条镜头都按这条规划。
- **mood / 光线 / 色彩**全程统一；机位多用推、拉、轻摇，保持电影质感。

---

## 六、喂给 Seedance 的静帧 / 场景图怎么搭（image 阶段）

- 场景图可以写得**极其细**：先用大模型把"情绪简介"扩成完整摄影语言，再出图。案例的城市图涵盖前 / 中 / 远景、机型与镜头（Arri Alexa + 变形宽银幕）、色彩分级，甚至给出一串 **HEX 色值**锁定调色，并加"无人物"之类硬约束。
- 角色母版图统一：全身、白底、柔和电影光——这是"模型每次比对的基准"。

---

## 七、安全 / 平台（与你套件口径一致）

案例原文含血腥与露骨内容，**搭 skill 时不要照搬**：

- 暴力 / 血腥一律转非直白——战损痕迹、暗红污痕、破损衣甲、不直视伤口的侧面构图、远景模糊倒影、以光影暗示创伤（与你道具 / 服装 / 分镜大师的"血迹 → 暗红污渍"完全一致）。
- 不写露骨 / 低俗台词。
- 角色定妆 / 资产沿用"不拿明星照 1:1 还原"的肖像合规原则。

---

## 八、对接你的六个 skill

- **分镜大师**出的"关键帧静帧提示词" → image2 出静帧 → 作为 Seedance 的**首帧 / 参考**。
- 新的"Seedance 视频提示词 skill"做的事：拿到一个关键帧（或一组同场镜头），输出一条 Seedance 视频提示词 =
  `@资产标记 + 摄影机行为 + 表演 / 物理 + （可选）多镜头切点 + （可选）台词 + 画幅`。
- @ 标记对应你 image2 做好的人物 / 场景 / 道具 / 服装资产。
- 画幅与分镜大师一致：开场问用户（横屏 16:9 / 竖屏短剧 9:16 等）。

## 九、原创示例骨架（我写的，非案例原文，按古装题材）

> 一条影视剧 Seedance 视频提示词的样子（首帧用 image2 的"金殿·君臣隔屏对峙帧"静帧）：

**【中文 Seedance 提示词】**
@杨延昭 立于 @金殿 殿心，手握 @断枪。镜头从屏风侧前方的低角度静止起幅，随他抬眼而轻微一震，快速推进到面部中近景；保持 @杨延昭 资产的脸型、发型与气质。表演：肩背端正、指节因隐忍而微微收紧，目光直刺屏风后的人影——克制而压抑的对峙。物理与力：殿门高侧硬光自左上切入，光柱里的体积浮尘随他起身被轻微搅动。随后硬切到屏风后人影的过肩反打，保持原场景的空间结构与光源方向一致。氛围肃杀、冷调青石灰为主，断枪暗红绳为颜色锚点。16:9 横屏，电影级写实，真实材质。避免：海报站姿、均匀轮廓光、全画面平均高清、网游仙侠发光、过度血腥。

**（可选台词）**："你也配问我？"——带克制怒意的低声。

## 十、提炼成一句话

Seedance 视频提示词 = **锁好资产（母版图 + @ 调用）** ＋ **把镜头运动、表演、物理、切点、台词写成现场导演语言**。静态长相靠资产保证，动态戏靠提示词把"力与节奏"写出来。
