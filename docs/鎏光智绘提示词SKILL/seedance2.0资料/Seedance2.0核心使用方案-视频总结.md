## 二、核心方法论概览

视频把「写好一段 AI 视频提示词」拆成四大能力，对应课件的四幕：

1. **时序拆解（叙事结构化）** — 把一段故事切成时间片段，逐段锚定要素。
2. **动作与运镜规范** — 用可观测的身体动作和单一运镜表达情绪。
3. **多参考能力** — 让形象 / 环境 / 动态 / 声场四类参考素材各司其职。
4. **编辑策略（延展·组接 / 局部·首尾帧 / 质量锚定）** — 后期精修与稳定输出。

## 三、分章节详解

### 第一幕 · 核心概念

#### 1. 时序拆解（THE FOUNDATION）
> 将叙事切割为时间片段，每个片段锚定 **主体、空间、动作、视角** 四要素。

- **反面示范（"非常假的"写法）**：
  "一位咖啡师在清晨的咖啡馆，有只猫趴在柜台上，正在冲煮一壶手冲。" —— 信息笼统、缺乏镜头感。
- **正确做法**：按时间切片，每段写清四要素：
  - 切片 A（0–5s）：主体=咖啡师 / 空间=晨光咖啡馆 / 动作=拿起手冲壶 / 景别=中景固定
  - 切片 B（5–10s）：主体=咖啡师 / 空间=吧台特写 / 动作=螺旋注水 / 景别=特写缓推
  - 切片 C（10–15s）：主体=咖啡师+猫 / 空间=整间咖啡馆 / 动作=擦拭壶身 / 景别=近景横移
- 最终形成一份完整的「镜头清单」。

#### 2. 正确写法案例（CASE STUDY）— 草原骑马 / 驯鹰
按 **0–5s / 5–10s / 10–15s** 三段分别写满整段提示词，示例要点：
- **0–5s**：中景固定镜头，辽阔大草原上天高云阔，男主身着蒙古族传统服饰，缓缓抬起右手拿起皮质护手套……口中念出"真正的自由，不是无拘无束"，一只雄鹰在高空盘旋。
- **10–15s**：远景固定镜头，一人一马在金色夕阳下化作一道剪影。
- 关键认知：**10–15 秒这种远景固定镜头是非常有掌控感的**。

#### 3. 动作的节奏（SLOW & CONTINUOUS）
> 让动作有层次、有衔接、有温度。用身体的细节传递情绪，而非抽象的情绪标签。

- **写小幅度的动作**：小幅度动作更容易被精准还原（缓步前行、轻抬手臂、微颔下颌、自然落座）。
- **承接余势**：从静止顺滑滑入下一个动作，让连续性自然流淌。
- **用身体语言替代情绪词**：不要在提示词里直接写"悲伤""愤怒"，而用唇角轻颤、眼尾渐润、肩头微耸等**可观测的身体语言**代替。

#### 4. 运镜规范（ONE SHOT, ONE MOVE）
> 引擎对中文镜头语言有深度理解，直接给出景别与运镜指令即可生效。

- **一个镜头一个运动**：推、拉、摇、移、环绕不要在一段里同时出现。
- **稳定优先**：先决定主体景别，再决定唯一的镜头运动，让画面呼吸感统一。
- 示例：远景缓推平移固定镜头。

### 第三幕 · 多参考能力（ACT III）

> 让参考素材各司其职，引擎才能精准复刻你的意图。

#### 四类参考素材（REFERENCE）
**推荐配比：形象参照 1–2 张 + 环境参照 1 张 + 动态参照 1 段 + 声场参照 1 段（总计 4–5 个素材）。**

| 类型 | 作用 | 要点 |
|---|---|---|
| ① 形象参照 | 锁定角色外观、服装、年龄感 | 通常 1–2 张就够，避免同一人物给太多冲突照片 |
| ② 环境参照 | 固定空间气质、光线与材质关系 | 不是"装饰图"，而是**场景稳定器** |
| ③ 动态参照 | 提供镜头轨迹与动作节奏 | 重点是学**运动轨迹**，不是复刻原视频的人物和场景 |
| ④ 声场参照 | 提供声音 / 配乐参照 | 配合整体氛围 |

> 提示：稳妥配置通常是 4–5 个素材，素材并非越多越好。

**实操演示**：用一段「男主在舞台中央演唱」的 15s 视频（Seedance 2.0 VIP），通过 0–5s / 5–10s / 10–15s 分段提示 + "男主要唱出这首歌"等指令完成生成。

### 第四幕 · 编辑策略（EDITING）

#### 1. 延展 · 组接（EXPAND VS COMBINE）

| 延展 · EXPAND（长镜头·沉浸） | 组接 · COMBINE（短镜头·冲击） |
|---|---|
| 对话持续推进、情绪层层递进 | 高速剪辑、视角频繁切换 |
| 单一空间内的连续位移 | 跨场景大幅度时空跳跃 |
| 追求一气呵成的沉浸体验 | 追求节奏张力与视觉冲击 |
| 适合 30–45 秒不间断叙事 | 拆分为 10–13 秒独立片段 |

- **核心原则**：**文戏优先"延长"，武戏优先"拼接"**。前者为了连续情绪，后者为了可控冲击。
- **延长提示词写法**：`将 @视频1 向后平滑延长 N 秒，保持角色外观、镜头方向、动作节奏和空间关系一致。`
- **实操演示**：把一段「老电影院两人奔跑」的视频延长 15 秒，电影悬疑风格、16:9 横屏、延续阴郁色调，并叮嘱 `no watermarks, no subtitles`。

#### 2. 局部与首尾帧（PRECISION / CONTROL）
> 在精确的时间坐标和空间区域内下达修改指令——让每一次调整都有迹可循。

- 替换人物、调整道具、修正画面 —— 在保持画面其余部分不变的前提下做"精准手术"。
- 善用**首尾帧锚定**，配合"帮我改一下这个人""要把时间表填几秒、画面停一层、改一遍"等局部指令。

#### 3. 核心约束 · 质量锚定语（THE GOLDEN RULE）
涉及人物出镜的内容，建议在指令**末尾附上一段质量锚定语**，放在句末用来"收紧"整段输出，而不是埋在中间被剧情信息冲淡：

> **"面部稳定不变形、五官清晰、人体结构正常、动作自然流畅、不僵硬、画面无卡顿、无闪烁"**

## 四、核心要点速记（Checklist）

- [ ] **拆时序**：按 0–5s / 5–10s / 10–15s 切片，每段锚定 主体·空间·动作·视角。
- [ ] **写小动作**：用可观测的身体语言代替"悲伤/愤怒"等抽象情绪词。
- [ ] **一镜一动**：每段只用一个运镜（推/拉/摇/移/环绕择一），稳定优先。
- [ ] **配参考**：形象 1–2 张 + 环境 1 张 + 动态 1 段 + 声场 1 段（共 4–5 个，不贪多）。
- [ ] **文延武拼**：文戏用"延长"保持连续情绪；武戏用"拼接"制造可控冲击。
- [ ] **加锚定语**：人物出镜的提示词末尾追加质量锚定语收紧输出。
- [ ] **要干净**：必要时加 `no watermarks, no subtitles`。
