## 一、万能提示词公式（核心结构）

> **主体 + 动作 + 场景 + 光影 + 镜头语言 + 风格 + 画质 + 约束**

按模块逐项填写，缺一不可：

| 模块 | 对应内容（示例） |
|---|---|
| **主体** | 一个男人（需保持角色稳定，可结合图生视频/参考图锚点） |
| **动作** | 撕裂腰带 → 肉体生化增殖 → 痛苦嘶吼 → 变身完成（10 秒内） |
| **场景** | 末日城市天台，战火纷飞，灰蓝色城市群，阴天 |
| **光影** | 低饱和灰蓝主调，紫色光效（从腰带核心炸裂），暗部压抑 |
| **镜头语言** | 一镜到底，环绕运镜，仰拍左侧 → 极速环绕至正面 → 环绕右侧俯拍，手持呼吸感 |
| **风格** | 暗黑特摄（BLACK SUN 美学），生化机甲，战损，痛苦美学 |
| **画质** | IMAX 胶片质感，Panavision 镜头，4K，颗粒感，边缘柔焦 |
| **约束** | 角色面部稳定不变形、人体结构正常、动作流畅不僵硬、画面无闪烁 |

**组合后即为一条可直接使用的完整提示词。**

## 二、分模块高阶写法

### 1. 动作描述（Seedance 最吃这个 ⭐）
- 核心是「**变身/动作**」，但不能只写「变身」二字，必须**分解为连续、细腻的细节**，确保 AI 生成流畅的物理变化。
- 按时间逐段分解（示例：0-2s / 2-4s / 4-6s / 6-8s / 8-10s 各写一个动作阶段）。
- **动作写作技巧**：
  - 使用「缓慢、猛然、持续、连续」等修饰词，避免「高速」「剧烈」等模糊词。
  - 动词组合要具体：撕裂、刺破、炸裂、闭合、震动、弹射。
  - 加入感官细节：痛苦嘶吼、骨骼脆响、电弧闪烁，提升真实感。

### 2. 镜头与运镜（直接提升电影感）
- 「**一镜到底环绕运镜**」是 Seedance 2.0 的强项，但需明确写出**运镜轨迹**。
- 运镜分解示例：开场仰拍左侧 → 变身中急速环绕至正面 → 变身后环绕右侧俯拍 → 起飞镜头极速后拉。
- 运镜关键词：
  - **景别**：特写、中景、全景。
  - **方式**：缓慢推镜、急速环绕、平稳横移、极速后拉。
  - **稳定要求**：手持呼吸感、随嘶吼震颤、画面流畅不抖动。

### 3. 角色稳定（防变脸 / 防变形）
> 角色需「参照上传图片」时，必须添加约束词防止 AI 自由发挥。
- **面部**：五官清晰、面部稳定、不变形、不扭曲。
- **人体**：人体结构正常、比例自然、动作不僵硬。
- **一致性**：同一角色、服装一致（初始服装为参考图片）、发型不变。

### 4. 画面质量（必加，提升清晰度）
- **清晰度**：4K、超高清、细节丰富、锐度清晰。
- **质感**：电影质感、IMAX 胶片质感、Panavision C 系列镜头（35mm, f4）、边缘柔焦、胶片颗粒。
- **稳定性**：无穿模、无重影、无闪烁、画面稳定。

### 5. 风格与氛围（快速出片）
- 写明风格关键词（核心词 + 色彩词 + 氛围词）。
- 示例：核心「暗黑特摄、BLACK SUN 美学、生化机甲」；色彩「低饱和灰蓝主调、暗部压抑、紫色光效」；氛围「末日废土、生物肉质与科技融合、恶心与痛楚感」。

### 6. 多镜头 / 分镜写法（Seedance 2.0 强项）
- 即使是 10 秒内的「一镜到底」，也可包含多个阶段。
- **按时间顺序描述**，避免碎片化。

## 三、15 秒短剧分镜结构（5 次运镜通用模板）

> 黄金结构：**15 秒 = 5 次运镜，每 3 秒一镜**。

时间轴：`00:00-00:03 / 00:03-00:06 / 00:06-00:09 / 00:09-00:12 / 00:12-00:15`

每一次运镜固定写清 **5 个要素**：

1. **运镜方式**（标题）：如「低角度仰拍推镜」「侧面横移跟拍」「特写急推 + 匹配剪辑」「环绕旋转升镜」「极速后拉 + 定格」。
2. **画面构图**：景别 + 起幅落幅（从哪到哪）。
3. **人物动作**：可观测的身体语言、微表情。
4. **对口型台词**：标注（角色，声线，方言，口型完全同步）+ 台词内容；无对白则写环境音。
5. **光影 + 运镜细节**：光的方向/色温变化、几秒内完成什么运动、焦点如何转移。

**情绪曲线**贯穿 5 镜，做几何倍数递增（例：卑微→屈辱→恨意→黑化→宣告）。

## 四、开头固定写法：画幅与质量标准

每条提示词开头先锁定技术规格（按题材微调风格词）：

```
竖屏 9:16，总时长 15 秒整，8K 超高清，60 帧高帧率，写实微电影质感，
杜比视界 HDR，广色域，<风格色调>，真实服装织物纹理（丝绸品反光、刺绣细节拉满），
光影层次丰富，无穿模、无畸变、无 AI 失真，口型与台词 100% 精准同步，
情绪、微动作与台词语境完全匹配。
```

## 五、保持一致性：固定角色 / 固定场景写法

在分镜前先「建档」，全程复用同一段描述：

- **固定角色**：年龄 + 身份 + 发型 + 妆容 + 服装 + 眼神气质，结尾统一加「**全程固定五官、发型、穿搭不变**」。
  - 同一人物多形态时（如素衣态 / 黑化态），用「**利用 Seedance 2.0 多模态参考图锚点，确保两张脸完全一致**」。
- **固定场景**：空间 + 背景陈设 + 光线 + **环境音**（如「烛火爆裂声、远处的更夫锣声」「空调低鸣声、键盘敲击声」）。

## 八、避坑指南（千万别写的内容）

- **复杂动作**：复杂多人互动、打架、跑跳剧烈、大幅度舞蹈 —— AI 难以稳定生成。
- **模糊描述**：「好看、很美、很酷」这类词无法有效引导 AI 生成。
- **矛盾要求**：超高速 + 极度稳定（AI 难以同时满足）。
- **违规内容**：色情、暴力、危险动作、版权人物。

## 九、可直接复制的万能模板（基于示例）

**模板：暗黑特摄变身短视频**

```
一个男人（参照上传图片）在末日城市天台，战火背景，灰蓝色调阴天。
他猛然撕裂腰带，引发肉体生化增殖：甲壳刺破皮肤，彩虹色碎片重组，
乳白色肌肉覆盖，蝗虫胸甲闭合，左胸盘炸裂展开半透明虫翼（布满血管）。
他仰天长啸嘶吼，面部被生物薄膜吞噬拼出昆虫面具，头盔闭合，复眼亮起淡紫光。
虫翼高频震动，垂直弹射起飞，空气撕裂，留下生物磷粉。
光影：低饱和灰蓝，紫色光效从核心炸裂。
一镜到底：左侧仰拍 → 急速环绕至正面再到右侧俯拍，手持呼吸感。

人体结构正常、动作流畅自然，10 秒内完成变身。
```

### 核心技巧总结（呼应示例）
1. **动作写慢、写连续**：把「变身/大动作」分解为撕裂、刺破、覆盖、闭合、展开、起飞等连续细节，避免跳跃。
2. **运镜写稳、写简**：明确轨迹（仰拍 → 环绕 → 俯拍），加「手持呼吸感」增加真实度。
3. **必加约束词**：面部稳定、人体正常、不变形，确保角色一致性。
4. **画质 + 风格词**：IMAX 胶片、低饱和灰蓝、暗黑特摄，快速定调。
5. **不写剧烈、复杂内容**：避免多人互动 / 高速运动，聚焦单角色变身即可降低生成难度。

> 一句话：**把任意创意脚本，按「主体+动作+场景+光影+镜头+风格+画质+约束」八要素拆开重写，动作分段、运镜单一、结尾加约束，就能转化为符合 Seedance 2.0 的高质量提示词。**
