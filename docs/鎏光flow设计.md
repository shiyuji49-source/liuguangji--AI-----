# 鎏光flow 设计文档

合并 image-studio（DMXAPI: gpt-image-2 / gemini-3-pro-image）与 video-studio（火山方舟 Seedance 2.0）为单一 Google-Flow 式生产工作台。基于对 Google Flow 的多源调研（官方 help/blog/labs + 文耘站 + 官方教程视频，4-agent workflow 交叉核验）。

## 1. Google Flow 制作界面 — 权威功能清单

| 功能 | 说明 | 我们能否做 |
|---|---|---|
| 统一提示词输入条（底部） | `@` 提及命名素材；承载模式/模型/设置/生成 | ✅ 已有 @资产 |
| 模式/模型选择器 | 点模型名切 图像/视频 + 子模式 | ✅ |
| Ingredients（参考素材，≤3 引用，需 @ 才生效） | 角色/物体/风格参考，跨生成复用锁一致性 | ✅ 各模型上限更宽 |
| Ingredients 抽屉（左） | 历史帧 + 参考图集合 | ✅ |
| 宽高比 16:9/9:16 | 提示框内切 | ✅ |
| 输出数 + 时长/分辨率 | 变体 1–4、片段时长、720p/1080p/4K | ✅ |
| History 版本栈（右） | 选中素材的历次版本 + 原始提示词，可恢复 | ✅ meta.prompt 已存 |
| 中心画布预览/缩放/下载 | 居中播放当前图/片段 | ✅ |
| Scenebuilder 时间线（底部） | 拼片段成连贯场景、trim、重排、导出 | ⚠️ 降级（Seedance 不能服务端拼接） |
| Frames-to-Video（首/尾帧） | 提供首尾帧补间 | ⚠️ Seedance first/last_frame，改为从资产墙选静帧 |
| 相机控制（运镜滑杆/预设） | Veo 原生运镜 | ⚠️ Seedance 无结构化相机 → 运镜词写入提示词 |
| Extend 延长 | 片段尾续接 | ⚠️ Seedance 无原生，v2 用末帧续接模拟 |
| Jump To 跳切过渡 | Gemini 生成片段间过渡 | ❌ Veo 专属 |
| Insert/Remove 视频物体编辑 | 片段内增删元素 | ❌ Veo 专属 |
| Refine 视频转视频编辑 | 自然语言改视频 | ❌ Seedance 不支持 |
| Voices 配音子模式 | 语音参考 | ❌ 配音走后期 |
| Flow Agent（对话编排） | AI 创意伙伴对话面板 | ❌ 违背非对话铁律 |
| Flow TV | 社区画廊 | ❌ 非生产功能 |

## 2. Flow 屏幕布局（四区 + 底部时间线）

- CENTER 中心舞台：当前选中 图/片段 预览、缩放、播放、下载。
- LEFT 左素材货架：Ingredients 参考 + 项目资产；图像生成入口在左。
- BOTTOM 底部命令条：提示框 + 模型/模式/画幅/输出数/时长/相机/生成。
- RIGHT 右 History：选中素材的版本/变体栈 + 原始提示词。
- BOTTOM 时间线（视频/拼接模式）：片段缩略图横排、trim、重排、导出。

## 3. 鎏光flow 布局提案

```
┌──────────────────────────────────────────────────────────────────────┐
│ 顶栏  ◀返回  项目名  画幅9:16·真人·悬疑   [⛶画布全屏]                      │
├───────────────┬─────────────────────────────────┬────────────────────┤
│ 左·素材区        │ 中·舞台画布                       │ 右·History 版本栈     │
│ Tab:参考/资产墙  │ 选中图/视频大图预览·点击放大·播放    │ 历次生成+原始提示词    │
│ [+上传新参考]    │ [下载][设首帧][设尾帧][设为参考]     │ hover 恢复/删         │
│ [从资产墙选]     │                                 │                    │
│ 按集/角色/场景过滤│                                 │                    │
│ [导入提示词生成器▾]│                                 │                    │
├───────────────┴─────────────────────────────────┴────────────────────┤
│ 底部·命令条 [图像|视频][image2▾][9:16▾][2K▾][×2变体][📷运镜▾] @道具…[生成]  │
├──────────────────────────────────────────────────────────────────────┤
```

### 关键设计
- **参考两来源严格分离**（题目③④）：左栏顶 Tab A「参考图（本次生成用）」含 `[+上传新参考]`（引入新像素，multipart）与 `[从资产墙选]`（复用已生成产物，复用 video-studio refPicker）；Tab B「资产墙」= 项目全部产物网格，按 集/角色/场景过滤。两个按钮不同图标/文案，杜绝混淆。
- **运镜=文本注入**：相机预设按钮把中文运镜短语追加进提示词（Seedance 无结构化相机），UI 诚实标注，不做 Veo 假滑杆。
- **导入提示词生成器**（题目⑨）三级钻取：选剧本 → 按集/按角色/按场景（同一份 shots/prompt_items 的三种 group-by 视图）→ 导入静帧(填图像模式)/视频片段(填视频模式+自动加 @资产母版进参考槽)。
- **点击放大预览 + 下载**（题目⑧）：中心 lightbox，图放大、视频播放器，下载按钮。
- **序列条 = 降级 Scenebuilder**：只读顺序预览 + 顺序导出（Seedance 不能服务端拼接/转场，合成交后期/P3）。
- **画幅默认 9:16**（短剧竖屏主力）。

## 4. 实现要点
- **复用不重写后端**：`/api/image-studio/*`、`/api/video-studio/*`+tasks、`/api/assets/[...path]` 全保留；新工作台 `src/apps/liuguang-flow/` 是前端聚合层。
- **并发守卫**：图像变体 >1 触发已知 M5（缺全局流控可超扣积分），上线前必须加全局并发上限。
- **能力差异显式提示**：Seedance 2.0 不支持真人脸参考图（真人项目 banner）；各模型参考图上限不同（动态显 已选 N/上限）。
- **黑金体系 + 非对话铁律**：沿用 .app-frame/.gold-chip/.card-generating/.text-liuguang；全程无对话框、无 Agent 面板。
- 读秒复用 `src/apps/prompt-studio/stopwatch.tsx`。

## 5. 分期实现
- **Phase 1 骨架**：liuguang-flow 应用 + 四区布局 + 底部命令条（图像模式：模型/画幅/档位/提示/生成+读秒）+ 中心画布预览 + 左资产墙(集/类过滤) + 右 History。图像端到端跑通。registry 合并入口。
- **Phase 2 参考与导入**：左 Tab 分离 上传新参考 / 资产墙选（③④）+ 点击放大下载（⑧）+ 导入提示词生成器 集/角色/场景 三级（⑨）。
- **Phase 3 视频与序列**：命令条视频模式 + Seedance 生成 + 首/尾帧选静帧 + 运镜文本注入预设 + 底部序列条（只读）+ 变体数并发守卫(修 M5)。
- **Phase 4 打磨**：History 版本栈完善、退役旧两个工作台、整体打磨。
